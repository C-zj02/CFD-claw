# 2026-05-20 网页端 LaTeX 公式渲染

对应提交：

```text
67fe444 Render LaTeX in web chat messages
```

## 改动背景

飞行器总体设计回答中会频繁出现工程公式，例如失速速度、升阻关系、翼载、推重比、航程和续航估算。如果网页端只显示纯文本公式，可读性较差。

## 支持范围

当前实现支持：

- 行内公式：`$...$`、`\(...\)`。
- 块级公式：`$$...$$`、`\[...\]`。
- 分式：`\frac{a}{b}`。
- 根号：`\sqrt{x}`。
- 上标、下标：`C_{Lmax}`、`V^2`。
- 常见希腊字母：`\rho`、`\alpha`、`\lambda` 等。
- 常见运算符：`\le`、`\ge`、`\approx`、`\times` 等。

同时避免代码片段中的 `$...$` 被误渲染。

## 技术实现

该提交没有引入外部 CDN 或完整 LaTeX 引擎，而是在前端实现轻量解析：

- 先识别代码片段，保护代码内容。
- 再识别行内和块级数学表达式。
- 将常见 LaTeX 命令转换为 HTML 结构。
- 用 CSS 控制分式、根号、上下标和块级公式样式。

## 验证方式

```bash
python -m py_compile src/web/app.py
python -m pytest tests/test_web.py
```

预期：

- 公式显示为数学样式。
- 普通 Markdown 仍能正常显示。
- 代码块中的 `$` 不触发公式渲染。
