# 26.1.7 复合连接安装

# 26.1.7.1 定义

复合连接安装是指同一零件(或同一组件)采用柔性连接和刚性连接相结合的连接安装方法。例如歼击机的座舱盖，两侧采用如图26.12所示的柔性连接，而前后弧玻璃开有花槽，用螺栓与骨架相连接而构成复合连接安装。

# 26.1.7.2 工艺方法

透明件复合连接安装的工艺方法包括以下内容：

# 1 座舱盖的初装

1）透明组件前后弧间隙的制取

前后弧间隙的制取过程是：

(1)舱盖骨架准确地装入装配夹具，固定夹紧。  
(2)用工艺通条由小到大，穿入涤纶带内，将玻璃固定在骨架上。  
(3)在玻璃件上划出前后弧框处的热间隙位置，取下透明组件进行修整。

2）前后弧框处花槽孔的制取

前后弧框花槽孔的制取过程是：

(1)用样板划出前后弧框花槽孔的位置。  
(2)按产品图纸要求制出花槽孔。

# 2 座舱盖的总装

1）密封物的敷设

按产品图纸要求在骨架前后弧框处敷设二层密封物，如XM17腻子布。

2）透明件两侧的固定

将粘有涤纶增强丙烯酸酯塑料加强件和涤纶带的透明件放入骨架，调整好位置，将涤纶带引入型材，然后用通条可靠地固定在骨架上。

3）前后弧框外蒙皮的安装

按产品图纸和技术条件的要求，安装前后弧框处外蒙皮。

3 交点的安装

在检验夹具上按产品图纸和技术条件要求安装锁钩和后交点。

4 涂密封胶和退火。

座舱盖安装完毕后，安装周边涂以XM18或XM23密封胶，在室温下硫 $24\mathrm{h}$ 后(硬度为邵尔 $A$ 型约30)装配件按退火规范退火处理。

# 26.2 密封装配

# 26.2.1 密封剂的要求

# 26.2.1.1 工艺性要求

对密封剂的工艺性要求有以下几点：

密封剂在施工时应具有流动性、塑性和触变性，按需要可刷、可涂和可注等。施工后有良好的成膜性。  
2)有足够的施工期，在施工时不流淌，具有一定的堆砌性。  
3)通常不需要加温硫化，而要求室温硫化。  
4)不干性腻子应具有可塑性，即腻子质地柔软，适于机械挤出或手工揉搓成所需要的形状。  
5)不干性腻子也应具有可拆卸性，即极易拆卸修理或更换。

# 26.2.1.2 施工环境要求

除另有规定，施工环境温度为 $15\sim 30^{\circ}\mathrm{C}$ ，空气的相对湿度控制在 $40\% \sim 80\%$ 范围内。环境应清洁、无灰尘、无油雾、无酸性气体和碱性气体。具备防尘和油污污染的措施。

# 26.2.3 密封工艺

# 26.2.3.1 密封前准备

# 1 密封处的表面准备

密封处的表面准备工作包括以下几点：

1)结构涂胶面上的灰尘、金属屑等，应用毛刷、吸尘器清除干净，但不允许用压缩空气吹拂。  
2)用汽油、丙酮、乙酸乙酯等溶剂湿润(但不过量)的擦布擦去涂胶面的油污和化学污垢。对有机玻璃涂胶面只允许用汽油清洗。  
3)清洗合格 $15\mathrm{min}$ 后应立即涂敷密封材料，放置时间过长则应重新清理。

# 2 非密封区的保护

为防止密封剂对非密封区的透明件表面污染，密封施工前应用医用胶纸带（或医用胶布）或其他适用的保护物保护，使密封区与非密封区隔开。施工整理后清除保护物。

# 26.2.3.2 密封用工具

密封时常用的工具有压注枪、平的或带锯齿形的刮刀、绘图毛笔、毛刷等。

# 26.2.3.3 刮涂密封法

在较大的贴合面上涂敷室温硫化密封剂时，可用压注枪把密封剂压注在表面上，用刮刀将密封剂沿表面刮成一定厚度的胶层，然后贴合组合件各部分。

# 26.2.3.4 刷涂密封法

在较小的贴合面上或连接件端头部使用膏状密封剂时，可用绘图毛笔蘸膏状密封剂涂敷于需密封处。

# 26.2.3.5 垫片铺贴与腻子填充密封法

# 1 密封布的铺设

密封布的铺设要求有：

1) 密封布的宽度应比透明件安装部位的宽度大 $2 \sim 3\mathrm{mm}$ 。  
2) 密封布在铺贴前必须去掉保护层，按产品图纸规定层数和部位铺贴。  
3) 密封布不允许搭接, 只允许在长度方向上无间隙的对接或斜接。  
4) 密封布应紧密贴合在透明件安装部位的表面上, 不允许有皱折现象。

# 2 腻子条的铺设

密封腻子条的铺设方法下如下：

1)将腻子预热到 $25\sim 30^{\circ}\mathrm{C}$ ，用腻子挤出器或手工揉搓制成直径为 $2\sim 3\mathrm{mm}$ 的腻子条。  
2)按产品图纸或技术条件的要求，把密封腻子条铺设于已铺有密封布的表面上。在连接安装过程中透明件端面与骨架之间充填密封腻子，确保气密性。

# 26.2.4 质量要求与检验

# 26.2.4.1 质量要求

透明件安装密封质量要求有以下几点：

1) 透明件安装密封部位的外表面蒙皮与透明件上不应有残留的密封物。  
2) 透明件安装密封部位的内表面密封区的密封物应整洁, 形状规则。  
3)飞机经抗压试验、气密试验无泄漏现象。

# 26.2.4.2 检验

# 1 密封工作前的检验

1) 使用的密封剂(物)应符合产品图纸或技术条件的要求, 使用的密封剂(物)必须是合格品, 否则不能使用。  
2) 密封工应经过技术培训合格。

# 2 密封过程中的检验

1) 密封部位必须是清洁干净的。  
2)按产品图纸或技术条件要求，检验密封物铺设的正确性。  
3) 密封部位若用膏状密封剂, 不允许漏涂或缺胶, 不允许出现气孔、缺陷和异物夹杂。  
4)密封部位若用膏状密封剂，不允许向非密封区流淌及有多余的密封剂。

# 3 最后检验

1) 透明件安装部位的飞机外表面及透明件上不应有残留的密封物。  
2)透明件安装部位的飞机内表面密封区应整洁、密封物的形状应规整。  
3)抗压、气密试验没有漏气现象。透明件无银纹、裂纹等缺陷。连接件无松动现象。

# 第27章 玻璃层合透明件工艺

# 27.1 概述

玻璃层合透明件是非常重要的一类飞机透明件，一般用作飞机风挡玻璃。

玻璃层合透明件是由玻璃、中间层材料、透明塑料板以及透明导电元件组成的复合结构。通过选用不同的材料、变化组合结构形式等技术措施可以使玻璃层合透明件达到强度、光学质量、耐环境稳定性等性能要求。

玻璃层合透明件按其结构材料可分为两类：全玻璃层合透明件和玻璃/塑料层合透明件。

飞机用层合透明件除上述两类外，还有全塑结构层合透明件，即由有机玻璃、定向有机玻璃和/或透明聚碳酸酯板用中间层材料粘合而成的全塑料板材复合结构的层合透明件。

玻璃层合透明件按功能和用途大致可分为安全风挡玻璃、抗鸟撞风挡玻璃、电加温风挡玻璃和防弹玻璃等四种。

# 27.2 玻璃板的加工与增强

玻璃具有良好的光学性质和机械强度，它具有的抗磨损性、耐老化性及耐环境稳定性是任何透明塑料无法比拟的。因此，玻璃是最重要的飞机透明材料之一。

按照飞机风挡玻璃图纸和技术条件的要求，玻璃原片在层合前要经过切裁、磨边、研磨和抛光等冷加工以及物理钢化或化学钢化增强处理等加工工序。

# 27.2.3 玻璃板的增强

# 27.2.3.1 方法、特点与适用范围

玻璃的理论强度很高，断裂极限在 $10000\mathrm{MPa}$ 以上。但是玻璃属于典型的脆性材料，表面微裂纹、内部缺陷和微观结构不均匀性等因素，使玻璃的实际强度仅为 $30\sim 80\mathrm{MPa}$

为了减小玻璃层合透明件的厚度、降低成品重量，同时又保证达到飞机风挡玻璃要求的强度，飞机用玻璃层合透明件所使用的玻璃板必须进行增强。通过增强处理，玻璃板的抗弯强度可提高 $3\sim 10$ 倍。

目前，实用的玻璃板增强技术有物理钢化法、化学钢化法和酸处理法。玻璃的几种增强方法及其增强机理、特点和适用范围等见表24.9。

# 27.2.3.2 玻璃板的物理钢化

# 1 物理钢化工艺过程及种类

玻璃的物理钢化工艺过程如图27.6所示。玻璃在钢化炉中被加热至高于应变点，接近软化点的温度，然后用冷风、液体或固体微粒等冷却介质淬冷，使玻璃表面形成残余压应力，从而提高其强度。物理钢化也叫做热钢化。根据冷却介质的不同又分为风钢化、液体钢化和微粒钢化。目前应用最广泛的是风钢化。

改变钢化工艺制度，尤其是冷却速度，可

以改变玻璃残余应力的水平，即达到不同的增强效果。根据增强的程度，钢化玻璃一般分成全钢化玻璃和半钢化玻璃两种。表27.10表明了全钢化玻璃和半钢化玻璃的应力大小、破碎状态和适用范围。

# 3 液体钢化和微粒钢化

液体的给热系数比空气大得多，所以用液体为冷却介质的液体钢化可以使玻璃达到更高的钢化程度及钢化更薄的玻璃。液体钢化使用的冷却介质是不与玻璃表面发生反应、不易燃烧

的液体，如硅酮、熔盐甚至是水等。

英国Triplex公司的专利技术生产的“1020”玻璃已应用在多种飞机风挡玻璃层合透明件上，“1020”钢化玻璃弯曲强度达到 $330\mathrm{MPa}$ （一般热钢化玻璃弯曲强度最高为 $186\mathrm{MPa}$ ），使用这种玻璃的飞机风挡玻璃厚度减小，重量降低，但仍可以达到结构强度如抗鸟撞击强度的要求。

流态化微粒固体介质的冷却效率可以达到风冷却的3倍，因此，微粒钢化 $2.5\mathrm{mm}$ 厚的玻璃的强度相当 $5\mathrm{mm}$ 厚风钢化玻璃。一般微粒钢化采用的固体介质是粒度为 $40\sim 80\mu$ 的 $\gamma-\mathrm{Al}_2\mathrm{O}_3$ 、 $\mathrm{Al}_2\mathrm{O}_3\cdot 3\mathrm{H}_2\mathrm{O}$ 和 $\mathrm{NaHCO_3}$ 等。

# 27.2.3.3 玻璃板的化学钢化

玻璃板的化学钢化工艺过程如图27.11所示。玻璃板浸渍于 $400^{\circ}\mathrm{C}$ 以上的硝酸钾等组成的熔盐中，熔盐中的 $\mathbf{K}^{+}$ （离子半径1.33Å）和玻璃中的 $\mathrm{Na}^{+}$ （离子半径0.97Å）间相互交换，由于离子占有容积的差别，玻璃冷却后在玻璃表面产生很大的压应力，可达 $45\sim 690\mathrm{MPa}$ 。玻璃的化学钢化又称为离子交换增强。玻璃网络结构对离子交换速度和深度有很大的影响，化学钢化的钠钙硅酸盐玻璃的离子交换层很薄，表面压应力不大，磨伤后强度下降很大，故飞机风挡玻璃层合透明件多使用铝硅酸盐玻璃，这类玻璃可达到很好的增强效果。

# 27.3 透明导电元件工艺

# 27.3.1 类型、特点和应用

飞机玻璃层合透明件上使用的透明导电元件主要是用作电热元件，带有这种透明导电元件的飞机电加温风挡玻璃，可以通电加温来防冰除雾，保证飞机正常飞行。而且，电加温可使胶合层保持一定的温度，使其处于最佳工作状态，提高风挡玻璃的抗鸟撞强度，并且可以消除冷态下高剪应力造成的脱胶和玻璃龟裂，延长风挡玻璃的使用寿命。

# 27.4 玻璃层合结构工艺

# 27.4.1 方法类型与特点

飞机风挡玻璃层合结构，其表层玻璃、一层或多层承力玻璃和/或透明塑料板，是用作为透明粘结剂的中间层材料将其粘结在一起的。这个层合工艺过程通常叫做夹层。

目前玻璃层合透明件用的中间层材料主要有：增塑和非增塑PVB中间膜、液态透明有机硅灌注料、聚甲基丙烯酸丁酯和甲酯胶片以及透明聚氨酸胶片和灌注料等。中间层材料的物理状态、工艺性质不同，要求采用不同的夹层工艺技术。玻璃层合透明件的夹层方法有：热压法（又称蒸压法）、灌浆法、叠片聚合法。这些方法适用的中间层材料、工艺持点见表27.22。

透明有机硅橡胶是一种具有良好的光学性能, 环境稳定性好, 使用温度范围 $-60 \sim +200^{\circ} \mathrm{C}$ 的耐高温的玻璃层合透明件中间层材料。目前国内可供使用的 GN-581 灌注型有机硅橡胶是乙烯基聚硅氧烷、甲基乙烯基硅树酯为基本成分的双组分无色透明的粘稠液体。使用前把两种组分按规定比例混合, 抽空排泡后灌注到两块玻璃板间预留的空腔中, 升温固化形成橡胶态的高透明的胶合层, 把玻璃粘结成为透明的层合件。

GN-581有机硅灌注料可用于玻璃板/玻璃板或玻璃板/有机玻璃板间的胶合。为了增加粘结力，通常使用GNB-593表面处理剂。

GN-581有机硅灌注料已应用于高速歼击机的前风挡玻璃层合件。

# 第28章 表面清洗、保护和修整

# 28.1 清洗

# 28.1.1 清洗目的和要求

有机玻璃进行清洗, 除去附着在表面的灰尘、油斑、残留的纸屑和胶痕等, 以便保持其原有的透明度、清洁度、以及良好的外观状态。有机玻璃板材在外观检查之前, 应除去保护纸或保护膜, 然后进行清洗。毛坯在加温成形之前亦需清洗。

# 28.1.2 清洗工作间

清洗工作间的主要技术要求有：

1) 工作间应有良好的通风和排风设施；  
2)地面保持清洁，一般为水磨石地面；  
3)门窗应严实，防止吹进灰尘；  
4)使用的压缩空气需经除油处理；  
5) 清洗工作间应与有机玻璃, 玻璃纤维增强材料等的加工间隔开, 防止静电吸附加工碎屑。

# 28.1.3 清洗剂种类

# 28.1.4 清洗方法

清洗有机玻璃的方法要点如下：

1)操作人员应穿戴无尘和无棉绒的工作帽和工作服；  
2)除去保护纸，用水冲洗有机玻璃表面，清除灰尘和纸屑；  
3)对难去掉的残留胶痕，可用蘸有无铅汽油的清洁脱脂棉轻轻擦去；  
4)用中性肥皂水(浓度 $2\% \sim 3\%$ )或清洗剂溶液(浓度 $5\% \sim 20\%$ )进行手工清洗，去除表

面的污垢；

5)用清洁的 $40\sim 50^{\circ}C$ 温水冲洗掉所用清洗剂的残液；  
6)用清洁的脱脂棉或软绒布轻轻擦干，亦可用清洁的压缩空气吹干。

# 28.1.5 清洗后质量检查

有机玻璃清洗后应进行质量检查。要求表面无灰尘、油污、胶痕、纸屑和其他杂质存在。用水膜法检查玻璃表面清洁度时，水应成线流下，不间断，表面不残留水珠。

检查结果如不符合技术要求，应再次进行清洗。

# 28.2 表面保护

# 28.2.1 表面保护的目的

有机玻璃表面硬度较低，为了防止其在加工制作、保管和运输过程中表面被划伤、擦伤或污染，必须粘贴保护纸或保护膜，或使用保护涂层加以防护。需要进行表面保护的场合有：

1)板材及制件在库房保管期间；  
2)板材及制件在运输过程；  
3)板材下料和制件机械加工过程；  
4)制件胶接时的非粘合区；  
5)制件组合或装配时的非组合装配区；  
6)飞机进行喷漆时。

# 28.4 特种表面涂饰

特种表面涂饰是指在有机玻璃及其制件表面涂覆一层透明涂层，赋予某种特殊性能，或者形成具有装饰性色彩，增加光泽和美观。

# 第29章 质量控制

# 29.1 概述

飞机座舱透明件是飞机上的关键部件，其质量直接影响使用安全性。为提高座舱透明件的质量，保证使用安全和使用寿命，对原材料、制造工艺以及产品性能进行全面质量控制非常重要。若其中任何一个环节质量控制不当或失控，都可能造成零件报废或隐藏某些质量隐患。这些质量隐患将会危及使用安全或缩短使用寿命。

本章主要叙述风挡、座舱盖等透明件制造过程中对工作人员素质的要求，并介绍对原材料、制造工艺及制件等的质量控制要求和方法。

应力-溶剂银纹性计算按下式：

$$
Y = \frac {3 P a}{b h ^ {2}}
$$

式中 $Y$ ——产生银纹的门坎应力值 $(\mathrm{MPa})$ ；

$P$ ——产生银纹时的力(N)；

$a$ ——支点 $AB$ 或 $CD$ 间距 $(\mathrm{mm})$ ；

$b$ ——试样宽度 $(\mathrm{mm})$ ；

$h$ ——试样厚度 $(\mathrm{mm})$ 。

报告结果是计算一组试样(不少于5条)的算术平均值、标准差和离散系数。

(2)悬臂梁法 该方法是美国和英、法等国采用的方法。MIL-P-8148E 和 MIL-P-25690B 中的试验装置如图 29.2 所示。

# 29.7.4.2 检验方法

制件光学质量的检测方法可分为两类，一类是目视法检查，另一类是采用仪器检测。目视法是最常用最方便的一种方法，但不能定量，且受人为因素影响较大。仪器检测在平面制件和曲面制件上均可采用，且可以定量，不受人为因素影响。下面简要介绍目视法检查和仪器检测光学角偏差和光畸变的方法。

# 1 目视法

目视法检查光学质量是将被检测的零件置于检验架上，检验人员在相当于飞行员座位上进行观察。观察视线与制件的最小夹角随飞机类型而异，对歼击机来说为 $30^{\circ}$ （见图29.8）。对于教练机的折返舱盖玻璃则要求更高，一般要求用 $15^{\circ}$ 检查。

# 第30章 安全卫生

在透明件的制造过程中，生产者亦会接触各种有害健康的化学物质和气氛、粉尘和易燃易爆物质，操作各种加热设备和机械、电气设备，同样需要了解有关的知识，熟悉、掌握和执行各种劳动保护法规、制度和措施，以保证生产安全和保护劳动者的健康。透明件生产中的劳动保护与玻璃纤维增强塑料(本书第17章)大体相同，本章仅结合透明件生产的特点简要介绍有关内容，其余可参考第17章有关部分。

# 30.1 作业环境

为保证航空透明件的制造质量，透明件制造车间应有较好的作业环境，特别是对制造中成形、胶接、装配等重要工序，更应有严格的环境要求。

# 30.1.1 环境条件

环境空气应干净、无油、无硬质尘埃。工作场地的空气中不应含有对材料和制件有害的溶剂蒸气。玻璃纤维、涤纶纤维加强件的机械加工应与成形和胶接工作间隔开。操作人员应穿戴清洁的工作服。

# 30.1.2 厂房温度

厂房一般应保持在 $15\sim 35^{\circ}\mathrm{C}$ 。成形间温度应保持在 $20^{\circ}\mathrm{C}$ 以上，冬季应备有加温装置。重要透明件的装配间应有保温和降温设施。

# 30.1.3 厂房湿度

厂房相对湿度一般应控制在 $85\%$ 以下。配胶及胶接工序必须在干净的室内进行，相对湿度不允许超过 $75\%$ 。

# 30.1.4 通风

厂房应有良好的自然通风条件和排气装置，胶接间应装有强力的抽风装置。成形间应采用双层密封门窗，以防止尘埃进入，并在成形过程中减少空气对流。

# 30.1.5 照明

厂房应明亮，并符合TJ33《工业企业采光设计标准》和TJ34《工业企业照明设计标准》的规定。

# 30.1.6 噪声

厂房应符合GBJ87《工业企业噪声控制设计规范》的规定，最大噪声不得超过90dB。

# 30.1.7 工作场地

工作场地应保持清洁卫生，设备和模具表面不应有油污、灰尘和其他不清洁物。应避免使用易产生静电的织物和地板，防止透明件吸附硬质尘粒或纤维。厂房地面应平整，清洗间应铺贴防滑的地面材料。厂房内应按消防要求布置消防设施。

# 30.2 劳动安全

机械和电气设备安全措施和防火防爆措施一般可参考本书第17章17.3.2.2和17.3.2.3。

美国采用的是阀限值 TLV(Threshold Limit value), 其基本含义与最高允许浓度相类似, 以每周接触有害物质 $40\mathrm{h}$ 期间的车间有害物质平均浓度表示 $(\mathrm{mg} / \mathrm{m}^{3}$ 或 $\mathrm{ppm})$ 。目前美国已规定出 600 多种有害物质的 TLV 值。

# 基础术语浅释

# 玻璃纤维增强塑料件部分

# 1 手糊成形 hand lay-up

在模具上用手工一边铺贴增强材料，一边涂刷树脂，直到所需厚度，固化（一般在室温和接触压下进行）后脱模得到制品。

# 2 模压成形 compression moulding

模塑料被约束在阴模与阳模形成的模腔中，在压力下固化成形的方法。

# 3 袋压成形 bag moulding

将片状预浸料铺贴在模具上制成坏料，覆盖一层不透气的柔性袋，用抽真空、加气压等方法，通过柔性袋传递压力至制件上，以达到加压固化成形的方法。

# 4 压力注射成形 pressure injection

亦称树脂传递模塑成形 resin transfer molding(RTM)。在一个阴阳对模的模腔中(模腔的尺寸与制件的尺寸一致)预先铺放增强材料, 合模后, 用压力将树脂体系注入模腔中, 使逐步浸润增强材料, 待树脂注满后, 密封模具, 固化, 脱模即得所要求产品。

# 5 纤维缠绕成形 filament winding

一种自动化成形方法。将经过预浸树脂体系的连续纤维(或纱带、布带)在控制的张力下按预定线型缠绕在芯模或模具上，然后固化成形。

# 6 拉挤成形 pultrusion

用牵引设备使预浸树脂的连续纤维(或纤维制品)通过具一定截面形状的加热模拉出, 然后经加热室进一步固化成形的方法。主要用于连续制造单向高强度型材。

# 7 现场发泡 foaming in place, site foaming

制造泡沫塑料夹层结构的方法之一。在已成形的壳体中注入配制好的组分或填入未发泡半成品，使之在壳体中发泡、固化成形。

# 8 层压板 laminate

两层或两层以上的薄片粘合在一起的板料。

# 9夹层结构 sandwich construction

在面板(蒙皮)中间夹轻质芯材的多层结构。常用的芯材有蜂窝芯、泡沫塑料、波纹板、空心微球等。

# 10 预浸料 prepreg

由树脂体系浸渍增强材料(纤维或织物)而成的片状材料。是制造纤维增强塑料的中间材料,可直接用于铺贴。

# 11编织套 contourwoven shape,contour woven fabric

亦称异形织物。是用增强材料编织的仿形织物，其形状与制件的形状相同。

# 12 基体 matrix

增强塑料中将增强材料粘结在一起的树脂相。

# 13 树脂体系 resin system

树脂与其它助剂形成的组合体。

# 14 树脂含量 resin content

增强塑料中所含树脂的数量，用所占质量或体积的百分数表示。

15 挥发物含量 volatile content

预浸料、树脂等含可挥发物的量，用质量的百分数表示。

16 凝胶时间 gel time

树脂体系在规定的条件下，由流动状态转化到具一定弹性的非流动状态(初始胶状)所需要的时间。

17 贮存期 storage life

材料在规定的条件下贮存，能保持所要求性能的期限。

18 适用期 pot life, working life

材料由配制好或从低温箱取出后能保持其适用性能(操作性能)的时间。

19 铺层 ply

形成层合增强塑料的每个单片材料（单层、单向带或织物）。

20 软模 elastomeric mold

一种具有弹性的软体模具，起加压或传递压力的作用。一般由膨胀橡胶制成。

21 吸胶材料 bleeder

用于吸收制件周围或表面多余树脂的疏松软质材料。

22 固化条件 cure cycle

使制件固化的温度、压力、保持时间和升温速率等条件的统称。

23 固化周期 cure period

制件固化时所需升温、降温、恒温、加压力和抽真空的全部时间。

24 预固化 precure

在制件整体最终固化前所进行的部分固化。

25 固化 cure

通过热、光、辐射或固化剂等作用，使热固性树脂经不可逆的化学反应进行交联的过程。

26 共固化 cocure

指增强塑料件与叠层坏料或叠层坏料与芯材，在一次固化中同时完成固化与胶接过程形成一个整体结构的工艺方法。

27 后固化 post cure

使热固性树脂增强塑料制件固化完全而进行的加温处理。

28 固化度 curing degree

表征热固性树脂固化的程度。一般用已固化的树脂占整个树脂的质量百分数表示。

29 热压罐 autoclave

一种密闭式可加温，采用惰性气体或空气加压，并同时对制件抽真空的压力容器型的成形工艺设备。

30 富胶区 resinricharea

树脂含量超过了允许量的区域。

31贫胶区 resin starved area

树脂含量低于最低限量的区域。

32 树脂堆积 resin pocket

在制件上形成的无增强材料的树脂结块。

33织物褶皱 fabricwrinkle

铺层在固化过程中形成的一层或多层永久性的隆起、凹陷或折痕。

34针孔 pin holes

已固化制件表面的小孔。

35 波纹 waviness

增强塑料件表面呈现起伏波纹状态。

36 气孔 void

制件由于层间裹入空气而形成的空隙。

37 格子凹陷 cell sink

夹层结构由于面板陷入蜂窝芯格中而使夹层板高度降低的现象。

38 蜂窝芯印痕 honeycomb core markoff

夹层板上所见六角形图案或线条。

39 架桥 bridging

一层或多层预浸料跨过圆角、台阶或蜂窝芯边缘时，在层间形成的空隙（未完全接触的状态）。

40 分层 delamination

层板中各层之间的分离或夹层结构中芯子与面板分离。

# 透明件部分

41 透明件 transparency

玻璃或透明塑料为主体制成的、用于飞机上观察和采光的制件统称。由透明平板件或成形件和为提供特种功能而设置的透明导电元件，以及为安装目的胶接于其上边缘连接件组成。

42 飞机座舱透明件 tsanspanencies for aircraft enclosure

飞机乘员舱上用的透明件，包括风挡、座舱罩、客舱舷窗和其他观察窗等。

43 成形件 formed parts

透明板材经加工成形，制成具有特定形状而尚未加有其他附属件的制件。

44 单层板 monolithic

一层透明塑料板或一层玻璃板。

45 层合透明件 laminated transparency

由两层或两层以上的透明塑料板或玻璃用柔性中间层粘合而成的夹层结构件。按其结构组成有玻璃层合件、透明塑料层合件和玻璃一塑料复合透明材料。根据用途不同分防弹玻璃、抗鸟撞风挡玻璃、电加温玻璃和航测玻璃等。

46 边缘连接件 edge attachment

透明平板件或成形件边缘与加强件或软固定带胶接组成的连接安装部分。

47 透明件材料 materials for transparecy

制造透明件用材料的统称。包括透明塑料和玻璃主体材料，透明中间层材料，透明涂层和用于边缘连接件的纤维增强塑料、合成纤维织物及胶粘剂。

48 透明塑料 transparent plastics

有机高分子化合物组成的透明材料，主要有浇铸有机玻璃、定向有机玻璃和聚碳酸酯板等。

49 有机玻璃 acrylic plastic sheet

又称“丙烯酸酯塑料板”。甲基丙烯酸甲酯单体为基聚合或共聚合改性制成的热塑性透明塑料板。

50 浇铸有机玻璃 cast acrylic plastic sheet

甲基丙烯酸甲酯单体中加入引发剂、耐光剂或增塑剂及其他组分制成的浆液，浇注于无机玻璃模具内，加温聚合成的有机玻璃板。

51定向有机玻璃 stretched acrylic plastic sheet

又称“拉伸丙烯酸酯塑料板”。浇铸有机玻璃板加热至软化状态，沿板平面方向拉伸或垂直于板面方向压缩，使聚合物大分子沿板平面取向，冷却后所得的板材。

52定向度 degree of orienting

又称“拉伸度”(degree of stretching)，定向有机玻璃拉伸取向的程度，以百分数表示。计算公式：

定向度 $S(\%) = (\sqrt{\frac{T_0}{T_1}} - 1) \times 100$

式中 $T_{0}$ ——拉伸（或压缩）前板材厚度

$T_{1}$ ——拉伸(或压缩)后板材厚度

# 53 聚碳酸酯板 polycarbonate sheet

一种透明的热塑性塑料板。由聚碳酸酯树脂经济出成形和压光制成。其特点是冲击强度极高，60年代开始用作飞机透明件材料。

# 54 钢化玻璃 toughened glass

又称增强玻璃，由无机玻璃经物理或化学强化处理制成。钢化使玻璃内部处于拉伸状态，表面产生一种强化的压缩层，以提高其弯曲强度和耐磨性。

# 55 物理钢化 physical tempering

又称“热钢化”(thermal tempering)。将玻璃加热至接近软化点，用空气或液体介质使其表面急速均匀冷却而使之具有钢化特性的一种工艺方法。热钢化玻璃可从低钢化度至完全钢化，处于中间状态为半钢化。

# 56 化学钢化 chemical tempering

又称“化学增强”(chemical toughening)。将特种成分的硅酸盐玻璃置于硝酸钾等组成的熔盐中进行离子交换，使其表面增强的一种工艺方法。

# 57 钢化度 degree of temper

钢化玻璃中内应力大小的一种量度，通常以玻璃中部单位长度的光程差表示。

# 58 中间层材料 interlayer

制造层合透明件用的透明柔性材料，用作不同或相同面板之间的胶粘剂、热补偿层和能量吸收介质。

# 59 PVB polyvinyl butyral

聚乙烯醇缩丁醛（缩写词）。

# 60 接触成形 contact forming

将透明塑料毛坯加热软化使之与模具型面贴合，冷却后获得规定形状制件的成形方法。

# 61 自由成形 free forming

在一个密闭的模腔上，将加热软化的透明塑料毛坯的周边夹紧密封，在其一侧充气压或抽真空，使之吹起或下拉至需要的形状，冷却后定形的成形方法。

# 62 真空成形 vacuum forming

借助抽真空使透明塑料毛坯贴模或自由下拉至规定深度，冷却后定形的成形方法。

# 63 吹塑成形 blow forming

模框气动自由成形的简称。借助充气压将气压室模框上的透明塑料毛坯吹塑至预定形状的成形方法。

# 64 回缩成形 snap back forming

又称“反贴成形”。在一个密闭的模腔内，用抽真空或充气压使加热软化的透明塑料毛坯拉伸至一定高度，然后推入凸模，释去真空或压力，使毛坯回缩反贴至模具上，冷却后定形的成形方法。

# 65 中温成形 warm forming

定向有机玻璃在低于其热变形温度下成形的一种工艺方法。其特点是材料热松弛较小，不会明显收缩变厚，尺寸稳定。

# 66 松弛成形 relaxation forming

定向有机玻璃毛坯在低于热变形温度下弯曲贴模，然后在夹持状态下升温并保持一定时间后冷却定形的成形方法。其特点是制件的内应力小，外形稳定。

# 67 退火 annealing

为减少透明塑料板在成形、加工和安装过程中产生的内应力所采取的一种可控的加热和冷却工艺方法。

# 68 热松弛 thermal relaxation

定向有机玻璃板在一定的温度和时间条件下受热后，板材表面尺寸收缩的相对百分数。用作测量定向有机玻璃定向度的试验判据，计算公式如下：

$$
\text{热松弛} R(\%) = (\frac{l_0 - l_1}{l_0})\times 100
$$

式中 $l_{0}$ ——加热前试样平面上标记线的尺寸

$l_{1}$ 一一加热后试样平面上的相应尺寸

定向有机玻璃受热完全消向时的热松弛值与定向度的关系如下式：

$$
S(\%) = (\frac{R}{1 - R})\times 100
$$

69 软化温度 softening temperature

热塑性塑料在规定的试验条件下测得的材料达到规定变形量时的温度。塑料常用维卡软化点(ISO 306, GB 1633), 航空有机玻璃习惯用拉祖尔金法(BHAM说明书No.214)或 $\mathbf{H}\Phi \Pi$ (物理问题研究所)法测定。

70 热变形温度 heat distortion tempsature

塑料耐热性能的一种量度。塑料试样在一定的载荷下，在等速升温的液体传热介质中，受简支梁静弯曲作用达到规定变形时的温度。测试方法按ISO75或GB1634。

71 玻璃化温度 glass transition temperature

无定形高聚物从玻璃态回复到高弹态或从高弹态转变到玻璃态时的转变温度范围的大致中点。

72 刚性连接 rigid joint

又称“硬固定”，指靠螺栓或压框等金属紧固件直接将透明件安装在骨架或机身构架上的连接方式。其特点是连接强度较高。

73 柔性连接 flexible joint

又称“软固定”，在透明件边缘胶接承载织物，再将其安装于骨架或机身构架上的连接方式。其特点是受力均匀，不易产生应力集中。

74复合连接 composite joint

指在一个透明件上，刚性连接与柔性连接结合使用的一种连接方式。例如某些歼击机座舱罩，两侧用柔性连接，前后孤用螺栓连接。

75 银纹 crazing

透明塑料表面上或表面内呈现出的银色光亮的网状细微裂纹。按其产生的原因分应力银纹、应力一溶剂银纹和溶剂银纹。透明塑料抵抗应力或溶剂而不产生银纹的能力称为抗银纹性。

76 应力银纹 stress crazing

在应力作用下，透明塑料在垂直于作用力方向出现的银纹。

77应力一溶剂银纹 stress-solventcrazing

在应力和溶剂或溶剂蒸气协同作用下，透明塑料上出现的银纹。由机械应力和溶剂引起的银纹呈有序状态，称应力一溶剂银纹；由内应力和溶剂引起的银纹呈无规状态，称溶剂银纹。

78 $\pmb{K}$ 值 $K$ value

透明材料的韧性或抗裂纹扩展能力的一种量度。测试方法按MIL-P-25690中4.6.5。

79 加热点 pit caused by heating

又称“耐热点”，有时称旋光点。有机玻璃板加热后表面上产生的点状缺陷统称。

80 旋光点 optical ratalory pit

有机玻璃加热后表面出现的发亮球形或泡形点状缺陷。

81 波纹 wave

有机玻璃表面凹凸不平的波纹状缺陷。硅酸盐玻璃和层合玻璃胶合层中轻微的透明线状不均匀体。

82较小的光学缺陷 minar optical defects

指透过透明材料及制件观察物体时，不会使物像严重失真的轻微缺陷，如杂质、小气泡或轻微擦伤等。

83 分层 delamination

指定向有机玻璃板在加工和使用过程中或在水分作用下边缘出现的层间分离现象。

84 雾度 haze

1. 透明塑料及制件内部或表面的混浊现象。以光束垂直透过试件后偏离入射光方向的散射光通量与透射

光通量的百分比表示。

# 85 光学角偏差 angular (optical) deviation

又称“角位移”，“光偏差”。光线透过透明件后折射光线与入射光线之间形成的夹角，用弧分表示。它与材料两表面的不平行度、光束的入射角和材料的折射率有关。角偏移使观察者看到的物体不在它真实的方向上，影像与真实物体间的偏移距离随观测距离的增大而增加。

# 86 光畸变 optical distortion

透过透明件观察物体时，其几何形状发生变形的现象。由透明材料的光学缺陷或表面波纹引起，随光束入射角的增大而加重。

# 87 折射位移 reflective deviation

透过表面平行的透明件观察物体时，因折射引起的物体位置改变。它与入射角和材料的折射率与厚度有关，不随观测距离而改变。入射角为 $0^{\circ}$ 时折射位移为零。

# 88 多置像 multiple imaging

因透明件内多次内反射而观察到同一物体多个影像的现象。厚制件及夜航飞机容易出现多重像问题。

# 89瞄准区sitting area

又称“仪器区”。座舱透明件上飞行员用以射击和投弹瞄准或观察外界时所用的主要区域。

# 90 非瞄准区 scanning area

又称“非仪器区”。座舱透明件上瞄准区以外的一般观察区域。
