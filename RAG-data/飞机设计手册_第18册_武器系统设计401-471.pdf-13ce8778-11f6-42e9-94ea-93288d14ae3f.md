# 第18章 武器及火控系统的校靶

# 18.1 校靶的目的和分类

# （1）校靶目的

通过校靶建立飞机轴线、各武器轴线（发射方向）和火控系统各瞄准装置瞄准线之间符合设计要求的正确的相对位置。

由于制造和安装误差，武器和瞄准装置相对飞机轴线的位置，在安装初始状态下一般不能满足设计要求。因此必须进行校靶，使其位置达到设计要求。

需要校靶的武器包括：航炮、火箭发射器和导弹发射架等；需要校靶的火控系统瞄准装置有：瞄准显示装置（平视显示器、瞄准具的光学显示器）、雷达天线、照相枪、头盔瞄准具、红外方位仪、激光测距器、速率陀螺（角速度测量装置）等。这些武器和瞄准装置相对飞机轴线的距离位置是确定的，校靶机构应保证他们在角位置（俯仰、方位和倾斜）上的调整，有的武器和瞄准装置在飞机结构中保证其安装精度要求时（如在生产飞机壳体的型架上给出符合精度要求的定位面），可以不进行校靶（如速率陀螺等），这些应由具体型号飞机的专门规范确定。

# (3）校靶的调整要求

在校靶时，武器和瞄准装置的调整一般利用其校靶机构来完成。校靶机构应保证足够大的调整范围，即能够校正由于飞机制造和武器、瞄准装置制造及其安装所引起的偏差。一般对航炮、火箭发射器的调整范围为 $\pm 30^{\prime}$ ，瞄准装置的调整范围为 $\pm 1^{\circ} \sim \pm 1^{\circ} 30^{\prime}$ 。同时应留有一定的校靶余量，以便在飞机使用寿命周期内，当飞机结构变化、武器和瞄准装置更换等情况下需要重新校靶时保证校靶机构有足够的校靶范围，校靶余量一般留 $\pm 30^{\prime}$ 。

校靶机构一般为机械装置，但瞄准显示装置和雷达天线等，也可以实施电气校正，即采用电气校正的方法将瞄准线对准到靶板相应校靶点上（雷达天线的电气校正见本册第15章15.1.6.4）。

无论校靶机构是机械的还是电气的，都必须在飞机、武器及火控系统的系统设计时就要论证清楚，并相互协调好，正确、合理地确定安装结构、校靶形式及校靶量。

# 18.2.2 靶板设计要求

靶板是靶图的固定设备，其结构应能牢固地固定靶图，并能使靶图在上下、左右、倾斜和垂直方向上进行调整，以便调整靶图使其相对飞机置于正确位置。

实际上，因为靶板一般都比较大，所以制造各方向可调的靶板框架，不仅结构庞大、复杂，而且使用维护也不方便。因此，在实际校靶中可不设计制造专门的靶板框架，而是在规定的距离上仅设立靶板(贴有白纸)，在靶板白纸上按校靶的需要，并根据靶图图纸现场绘制1:1的靶图，采用一些一次性的调整措施，使靶图相对飞机处于正确位置后即可进行校靶。

# 19.2 火控系统综合试验准备的一般要求

一般对新型号的研制而言，试验准备的周期比试验实施周期长得多。因此，做好火控系统综合试验（以下简称试验）准备是火控系统研制乃至飞机研制的关键环节。图19-1简明描述了试验的典型准备和实施过程，其中虚线框内表明了试验准备工作项目。图中三方面的准备工作包括：试验任务书、试验大纲及试验操作规程等试验文件的编制；仿真综合及系统测试软硬件环境的开发；试验场地及辅助环境的建立。一般要采用交叉并行的方法进行这三方面的工作，以利在有限的准备周期内搞好试验准备。如果试验准备的基础条件和开发周期允许，也可以采用以往较为简单的分散式火控系统地面交联试验准备工作中采用的基本“串行”的方法（图19-2)：先完成试验任务书、大纲的编制，然后以试验大纲为依据开发测试环境、准备场地。这种方法，尽管准备周期长，但开发测试环境和准备场地的依据确切、完善，准备中技术协调少，工作顺利。

# 19.2.2 仿真/综合测试系统开发的一般要求

仿真/综合测试系统的功能是为被测火控系统提供：

载机、目标机的飞行仿真，子系统仿真和输入输出接口数据的仿真；

将各子系统逐个综合成完整的火控系统的环境；

火控系统的系统功能和性能的测试环境；

处理、分析测试数据的环境。

# 19.6 火控系统与武器交联试验的一般要求

火控系统与武器的交联试验包括火控系统与非控武器、火控系统与可控武器的交联试验。火控系统与非控武器之间没有信号交联关系，因此检查非控武器的发射状态、发射控制等可结合火控系统综合试验完成，不需要非控武器参试；而可控武器与火控系统交联试验时，相关武器及其发射架要参试（不应带战斗部），而且要提供其专用测试设备。火控系统与武器交联试验的一般要求见表19-12。

# 19.7 火控系统机上地面试验的一般要求

# 第20章 航空武器及火控系统的试飞要求

航空武器系统作为作战飞机的一个重要组成部分，在设计定型阶段，必须和飞机一起进行有关飞行试验，以检验其是否和飞机相适应，并满足飞机研制总要求的各项指标。

# 20.1 飞行试验（试飞）的分类

根据不同的需要，航空武器系统的各个组成部分或各分系统在研制过程中，要进行不同目的的飞行试验。

# 20.1.1 设计定型试飞（或鉴定试飞）

由国家航空军工产品定型委员会授权的试飞鉴定单位对新研制或改进的武器/火控系统进行设计定型（或鉴定）试飞，以考核其功能和性能是否满足研制总要求。

# 20.1.2 适应性试飞

由国家航空军工产品定型委员会授权的试飞单位，对已经定型（或鉴定）的被试对象在指定的飞机上检查其装机接口的正确性、使用环境的适应性及主要指标的正确性进行飞行试验。

# 20.1.3 调整试飞

在设计定型试飞前进行，一般由设计定型试飞单位承担，也可由研制单位承担，但应由设计定型试飞单位派人员参与。通过调整试飞以便检查武器/火控系统操纵的完整性和有效性，并能早期暴露设计缺陷，得到修改完善。

# 20.1.4 科研试飞

对被试对象的研制方案可行性、主要功能和技术指标进行的研究性飞行试验，或对被试对象的缺陷及故障，或设计更改后的效果进行飞行试验。

# 20.2 调整试飞的要求和内容

# 20.2.1 目的

调整试飞是在飞机首飞后和设计定型试飞前进行，其目的是：将飞机各系统调整好并具备应有的功能；将设计上存在的重大缺陷基本排除；检查地面支援设备的实用性，以便顺利进行设计定型试飞任务。

# 20.2.2 要求

调整试飞应使武器/火控系统在飞机飞行性能的整个飞行包线的 $80\%$ 范围内处于正常工作状态。

# 20.2.3 调整试飞的内容

调整试飞的内容包括：

# （1）火控雷达

通过规定科目的飞行，检查雷达空空测距、跟踪稳定性、空地测距、地图测绘（真实波束、扩展的真实波束地图测绘和地图冻结）和信标叠加等功能，以及在巡航和机动飞行中雷达的工作情况；检查雷达主要功能是否正常，显示画面是否与所要求的指示一致，包括双杆和雷达控制板等的操作是否正确方便等，以及按不同情况分别在空中和地面使用不同方式，利用自检测检查雷达故障。

# (2）惯导系统

通过规定科目的飞行，检查惯导的自动程序导航、人工选择航路点和即时位置改航的功能，在返航中检查惯导和塔康校正功能。在机动飞行中，检查过载、滚转和振动等对惯导的影响。通过地面机务准备，检查不同方式的对准时间、惯导的自检测功能、人工和自动装定数据的能力。在这些检查中，验证其显示画面、操作和控制逻辑是否正确。

# (3）火控系统的电子对接试飞

在这一试飞过程中，应检查：

a. 数据装定和自检测能力

通过地面操作和空中试飞，检查任务数据人工装定和自动装定能力以及加电自检测、周期自检测和操作员起动自检测等的能力。

b. 空空攻击功能

在确定的飞行状态和飞机构型条件下，检查航炮和导弹利用双杆、多功能显示器周边键和其他控制器进入预编程所规定的攻击瞄准方式的能力；检查平显、多功能显示器的显示画面、操作和控制逻辑以及其他信号是否正确。

c. 空地攻击功能

在确定的飞行状态和飞机构型条件下，检查航炮、航箭、炸弹和其他武器利用双杆、多功能周边键和其他控制器进入预编程所规定的攻击瞄准方式的能力；检查平显、多功能显示器的显示画面，以及操作和控制逻辑的正确性。

d. 备份功能的检查

结合有关的试飞科目，对所规定的备份工作状态进行检查。

# (4）航空武器

a. 航炮对地攻击

在确定的飞行状态和飞行构型条件下，装填一定数量的炮弹，检查航炮的限时和余弹量，在对地面目标攻击中，分数次发射完毕，此时仅检查功能，不考核精度。

b. 模拟挂弹状态下的炸弹投放

在调整试飞中，一般不进行空中投放炸弹的考核，仅通过地面模拟试验来验证系统是否具有单投、双投、连投和应急投放的功能，检查显示画面和控制操作逻辑的正确性。

# 20.2.4 测试

为了能全面检查武器/火控系统及其设备的各项功能，飞机应是全技术状态。应该尽量利用机上的测试设备，在条件允许下，可装少量必需的测试设备，但不能影响对飞机各功能的检查。在武器/火控系统调整试飞阶段，应该加装总线记录器。

# 20.2.5 保障设施

保障有三部分内容：

（1）测试参数的处理设备；  
(2) 保障各系统正常工作（检测、数据装定、录像回放、场站通信和导航等设备）的地面保障设备；  
(3) 保障火控系统功能检查所需的专项外挂（例如导弹的训练弹和模拟弹，以及挂弹架和发射架等）。

# 20.3 设计定型（鉴定）试飞的要求和内容

按国家航空军工产品定型委员会审批的设计定型（或鉴定）试飞大纲要求，安排对一、二和三级产品进行飞行试验。在这些产品中有新研制的、已定型而进行适应性工程改进的产品和直接采用的定型产品，所以对它们的设计定型试飞要求也有不同。一般来说，对一、二级产品都要进行设计定型试飞；对三级产品则采用鉴定试飞。对于新研制的产品或是已定型而进行适应性工程改进较多的产品应进行设计定型试飞；对已定型产品或已定型而进行适应性工程改进较少，且不足以影响主要性能的产品可以进行适应性试飞。

# 20.3.1 目的

设计定型（鉴定）试飞的目的是：

（1）检验武器/火控系统及其设备的功能及性能（包括可靠性和维修性）是否达到设计使用要求，并满足研制总要求；  
（2）验证武器/火控系统及其设计的正确性和飞行作战的适用性；  
(3) 完成武器/火控系统各级成品试飞鉴定和适应性检验试飞；  
（4）鉴定配套的随机设备和综合保障设备以及检测仪器的适用性；  
（5）积累飞行使用经验和数据，为用户提供使用文件资料。

# 20.3.2 试飞的要求和内容

武器/火控系统的设计定型试飞将验证其是否满足研制总要求所规定的性能指标，考核其自主导航和辅助导航功能、空空和空地攻击功能、系统的各项精度指标、控制和显示以及系统的综合管理能力。主要试验项目包括：有关分系统（如雷达、惯导等）的试飞、火控系统的电子对接试飞、武器系统的精度试飞（靶试）和环境条件试飞，并给出相应的试飞报告。

火控系统电子对接试飞是火控系统的精度试飞，而下面一节的靶试是全机武器/火控系统的精度试飞。它们在武器系统已经通过环境适应性试飞和功能验证试飞，认为航空武器系统功能正常、逻辑关系正确、能适应飞机规定的各种飞行条件和无电磁干扰等条件后，才能进行。如果火控系统中的各分系统（如雷达、惯导、大气数据计算机、速率陀螺、无线电高度表和武器随动挂架等）尚未定型，则应进行单独的设计定型（鉴定）试飞，在其各有关参数满足研制总要求或已得到明确的试飞结果的基础上，才能进行火控系统的电子对接试飞。

电子对接试飞应按航空军工产品定型委员会审批的设计定型（或鉴定）试飞大纲要求，检验在飞行过程中，火控系统的输出对输入的实时性的响应。试飞中，应对整个过程的输入、输出和有关中间处理值进行记录，并根据测试的输入量真实记录，处理出相应数据结果，校核其是否满足研制总要求。

# 20.3.3 结论要求

在设计定型试飞结束后，鉴定试飞单位提出飞机的武器/火控系统的试飞总结报告。其内容为：

（1）试飞科目、起落架次、日期、飞行时间和天气情况；  
(2）试飞条件；  
(3) 试飞方法；  
（4）试飞有效测试数据，测试数据应整理成曲线或表格；  
(5）数据处理方法和应用公式；  
(6) 数据处理结果 (用表格或曲线形式);  
（7）各项功能完成的成功概率；  
（8）精度及分析；  
(9）对航空武器系统显示和控制性能评价；  
（10）对航空武器系统工作正确性、稳定性和可靠性的评价；  
（11）存在问题及建议；  
（12）飞行员评述意见；  
（13）对航空武器系统总评价意见。

# 20.4 研制单位的保障条件

对研制单位而言，为了保证设计定型（鉴定）试飞，应做到：

# 20.4.1 参试跟飞人员

各研制单位的参试跟飞人员应熟悉被试对象的工作原理、软硬件结构，能在现场分析和

处理问题，保证试飞顺利进行。

# 20.4.2 被试对象

研制单位应按GJB1362《军工产品定型程序和要求》以及HB6459《航空火控系统地面联合试验通用要求》，完成被试对象的地面试验，并和武器完成地面联试。通过调整试飞使被试对象达到冻结的技术状态，具备进入设计定型（鉴定）试飞条件。

# 20.4.3 技术资料

为完成设计定型（鉴定）试飞，研制单位应提供与被试对象有关的技术资料：

(1) 试飞要求 (试飞任务书);   
(2）使用维护说明书；  
(3) 技术条件、技术说明书；  
（4）接口控制文件；  
(5）地面联试技术总结报告；  
(6) 被试对象的合格证书;  
（7）武器/火控系统总体误差分配报告（或精度分析报告）；  
（8）武器的弹道资料；  
(9）电路图、接线图、线术束图、线束敷设图等；  
（10）其他与试验有关的资料。

# 20.4.4 培训

被试对象研制单位应对试飞单位有关参试人员（试飞员、地勤人员和试飞技术人员等）进行技术培训，使他们能熟练掌握系统工作原理、软硬件结构、信号交联关系、系统操作和使用维护等知识。
