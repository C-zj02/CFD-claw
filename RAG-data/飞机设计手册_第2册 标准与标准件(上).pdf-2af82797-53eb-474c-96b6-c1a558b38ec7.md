# 13.4 航空滚动轴承

航空滚动轴承的技术要求, 是在 GB307.1—84、GB307.2—84、GB307.3—84、GB308—84、GB309—84 等有关滚动轴承国家标准规定的基础上, 依据航空工业的特点, 对轴承的材料、游隙、摩擦力矩、润滑剂等提出了特殊要求, 编制了 GJB269—87《航空滚动轴承技术条件》, 进行了补充规定。也就是讲, 按 GJB269—87 生产的滚动轴承不仅应满足有关国家标准的规定, 而且还应同时达到该标准的要求。

# 13.4.2 航空滚动轴承的标记

不同组别的轴承有不同的技术要求，同一型号的轴承可能在不同组别内出现，因此在订货时或在图样上的标记，应注明轴承代号、航空滚动轴承技术条件代号和组别号。

例如：轴承代号为501065K，组别为第5组

订货和图样上标记： $\frac{501065\mathrm{K}}{\mathrm{GJB269 - 87 / 5}}$

# 13.4.4 5组轴承的静破坏负荷

本规定适用于在飞机操纵系统中呈摆动状态工作的5组轴承，如用于操纵系统的摇臂、滑轮及舵面悬挂接头内的轴承。

(1)在下列工作条件下，使轴承破坏的负荷作为静破坏负荷（径向静破坏负荷）。

a. 轴承压入刚性壳体内(壳体外径比轴承外径大2倍的钢轴承座)；  
b. 轴承的环境温度为 $15^{\circ} \mathrm{C}$ ;  
c. 在轴承的内圈或外圈以 $15^{\circ} \sim 20^{\circ}$ 的幅度和 1s 的周期摆动时, 在 $15\mathrm{min}$ 内, 负荷由零不断增加到使轴承破坏的数值 (即在轴承破坏前大约发生 90 次摆动)。

(2)轴承的径向静破坏负荷(N)

$$
P _ {\mathrm {j p}} = C \cdot d _ {\mathrm {q}} ^ {2} \cdot Z \tag {13-1}
$$

式中： $d_{\mathrm{q}}$ —钢球直径 $(\mathrm{mm})$

$Z$ ——单列钢球的数量；

$C$ ——与轴承结构和尺寸有关的系数（ $\mathrm{MPa}$ ）。

$C$ 值见表13-28。

# 13.5 滚动轴承的选用原则

# 13.5.1 滚动轴承类型的选择

各种类型的轴承有着各种不同的特性，以适应各种不同使用条件的要求。选用轴承时，通常应根据下列主要因素综合考虑。

# (1)允许空间

在进行机械设计时，一般先确定轴的尺寸，然后再根据轴的尺寸选择轴承。若机构在径向空间受到限制，必须采用径向剖面高度较小的轴承时，可选用滚针轴承、特轻或超轻系列的轴承。若机构的轴向空间受到限制，必须采用宽度尺寸较小的轴承时，可选用窄系列或特窄系列的轴承。

# (2)轴承负荷

# a. 负荷大小

负荷大小是选用轴承最主要的因素。通常，球轴承适用于承受轻、中负荷，滚子轴承适用

于承受重负荷。相同类型的轴承，可视负荷的大小选择不同的尺寸系列。

# b. 负荷的方向

承受纯径向负荷时，可选用深沟球轴承、圆柱滚子轴承及滚针轴承。

承受纯轴向负荷时，可选用推力球轴承或推力滚子轴承。

当既承受径向负荷又承受轴向负荷(即联合负荷)时，一般选用角接触球轴承或圆锥滚子轴承。如果考虑装配时对中心性，可选用调心球轴承、调心滚子轴承。

# (3)轴承精度

对于大多数机械而言，选用G级和E级公差的轴承已足以满足功能要求。但对于旋转精度比较高的场合，如机床主轴、精密机械、仪表等，应选用高精度的轴承。高速旋转的机械亦应选用高精度的轴承。如航空、航天、航海陀螺仪轴承和航空发动机主轴轴承。

# (4)轴向游动

通常轴承的配置方式是在轴的一端采用一个轴向固定的“固定轴承”；而在轴的另一端采用一个轴向可以自由移动的“游动轴承”，以适应由于热胀冷缩而引起的轴的伸缩。当采用深沟球轴承、调心滚子轴承等非分离型轴承作“游动轴承”时，外圈一般采用间隙配合安装，使外圈与壳体孔之间能够在轴向自由移动。

# (5)摩擦力矩

影响轴承摩擦力矩的因素很多。一般而言，球轴承的摩擦阻力较滚子轴承为小，当要求低摩擦力矩时，应尽量避免采用接触式密封结构，因为接触式密封的摩擦力矩远远大于轴承本身的摩擦力矩。同时，建议采用油雾润滑、汽油润滑或其他有利于减磨的润滑方式。

# 13.5.2 滚动轴承的尺寸选择

轴承的尺寸选择通常根据所承受的负荷(大小和性质)、轴承寿命及可靠性的要求而定。

静止或缓慢摆动、转动 $(n \leqslant 10\mathrm{r / min})$ 的轴承承受的负荷为静负荷，快速旋转（ $n > 10\mathrm{r / min}$ ）的轴承承受的负荷为动负荷。

轴承的负荷能力用额定负荷来表示。各类轴承的额定动负荷及额定静负荷值可在轴承样本中查得。

# 13.5.3 摆动轴承的选用

对于滚动轴承的选用原则，在13.5.1和13.5.2中已阐述。选择的方法不外乎计算法和类比法，而最常用的方法是类比法，因为这种方法经济、方便、可靠(也称比照法)。

摆动轴承是飞机操纵系统专用的轴承，即航空滚动轴承中的5组轴承，也称为“小游隙轴承”，这类轴承无需要求旋转精度，但制造精度必须保证，安装后要保证转动灵活，无滞涩现象。在灵活性方面，最好是采用供需双方都可接受的标准样件。

飞机摆动轴承的选择可参照HB/Z203--91规定。

# (2)轴承固定的技术要求

a. 设计壳体时, 必须考虑到收口尺寸及冲头直径尺寸。

e. 在已经表面处理壳体上滚压或冲点的地方, 要涂防锈底漆 F06-8。

f. 对 A、B、C 三种固定方法, 必须经过试件验证试验, 以保证所使用的工艺在表 13-43 所规定的轴向载荷作用下, 不被压出。

g. 采用经过验证的工艺方法固定的轴承, 不进行压出试验。

# 13.7 滚动轴承钢球(GB308-84)

本标准适用于滚动轴承钢球和商品钢球。

# 13.7.1 定义、符号及解释

(1)球的公称直径 $D_{\mathrm{w}}$ 一般用于鉴别球尺寸的直径值。  
(2)球的单一直径 $D_{\mathrm{w8}}$ ——与球轮廓面相切的两平行平面间的距离。  
(3)球的平均直径 $D_{\mathrm{wm}}$ ——球的实际最大和最小单一直径的算术平均值。  
(4)球直径变动量 $V_{\mathrm{DWS}}$ ——一个球的实测最大和最小单一直径之差。  
(5)球形偏差 $\Delta S_{\mathrm{ph}}$ ——在任一径向平面内，球轮廓面的外接球体与球轮廓面上任一点之间的最大径向距离。  
(6) 批——制造条件被视为相同的一定数量的球，认作一整体。  
(7) 批平均直径 $D_{\mathrm{wmL}}$ ——同批中最大球的平均直径与最小球的平均直径的算术平均值。

$$
D _ {\mathrm {w m L}} = \frac {D _ {\mathrm {w m} \max} + D _ {\mathrm {w m} \min}}{2}
$$

(8) 批直径变动量 $V_{\mathrm{DWL}}$ ——同批中最大球的平均直径与最小球的平均直径之差。  
(9)球的等级——尺寸、形状、表面粗糙度和分选公差的特定组合。球的等级用数字表示。  
(10)球规值 $S$ ——批平均直径与球公称直径的相差量, 此量为一确定的系列。每一球规值为该等级已定球规值间距的整倍。球规值和球的等级与公称直径组合在一起, 应视为订户订货用的最严密的尺寸规格。  
(11)球规值的偏差 $\Delta S$ ——批平均直径减去公称直径与球规值之和。

$$
\Delta S = D _ {\mathrm {w m L}} - \left(D _ {\mathrm {w}} + S\right)
$$

(12)球分规值——最接近球规值的实际偏差的已定系列的诸量。每一分规值为该等级球已定球分规值间距的整倍。球分规值和球的公称直径与球规值组合在一起，为制造厂标志批平均直径之用，不应作订户订货之用。

# 13.7.3 钢球的标志

钢球的标志由下列各项按顺序排列表示。

(1)——钢球公称直径，用规值为零的直径值表示，单位 $\mathrm{mm}$ （不标出）。  
(2)——钢球等级，用符号 $\mathbf{G}$ 加数字表示。  
(3)——球规值或符号 $b$ (不按批直径变动量分组标志)。球规值用正负号加数字表示, 单位 $\mu \mathrm{m}$ (不标出)。

(4)——球分规值，用小括号内的正负号加数字表示，单位 $\mu \mathrm{m}$ （不标出）。  
(5)——表示本标准编号或本标准编号加补充技术条件编号。

钢球标记示例：

例1：材料为高碳铬轴承钢，公称直径 $8\mathrm{mm}$ ，等级10，规值 $+4\mu \mathrm{m}$ ，分规值 $-0.2\mu \mathrm{m}$

$$
8. 0 0 0 0 G 1 0 + 4 (- 2) \quad G B 3 0 8 - 8 4
$$

例2：材料为高碳铬轴承钢，公称直径 $12.7\mathrm{mm}(1 / 2\mathrm{in})$ 等级40，规值0，分规值0：

$$
1 2. 7 0 0 0 G 4 0 \pm 0 (\pm 0) \quad \text {G B} 3 0 8 - 8 4
$$

例3：材料为高碳铬轴承钢，公称直径 $45\mathrm{mm}$ ，等级100，不按批直径变动量、规值、分规值提供：

$$
4 5. 0 0 0 0 G 1 0 0 b \quad G B 3 0 8 - - 8 4
$$

# 13.8 滚动轴承滚针(GB309—84)

本标准适用于滚动轴承滚针和商品滚针。

# 13.8.1 定义、符号及解释

(1)滚针公称直径——用于标识滚针直径的值（图13-30、图13-31）。  
(2)滚针单一平面平均直径 $D_{\mathrm{wmp}}$ ——在单一径向平面内，实测最大与最小单一滚针直径的算术平均值。  
(3)滚针公称长度 $L_{\mathrm{w}}$ ——用于标识滚针长度的值。  
(4) 滚针单一平面内直径的变动量 $V_{\mathrm{DWP}}$ ——在单一径向平面内, 实测的最大与最小单一直径之差。  
(5)滚针单一直径 $D_{\mathrm{w}}$ ——在垂直于滚针轴线的平面（即径向平面）内与滚针表面相切又相互平行的两条切线间的距离。  
(6)滚针分组——用来限定滚针直径偏差的范围。在滚针长度中部的径向平面内，滚针平均直径 $D_{\mathrm{wmp}}$ 偏离公称直径 $D_{\mathrm{w}}$ 的上、下偏差。

例：-2/-4。

(7)圆形偏差——滚针表面和一径向平面的交线上的任意点与其外接圆之间的最大径向位移。  
(8)分组批——同一等级，同一公称尺寸，平均直径在同一分组内的一批滚针。  
(9)分组批直径变动量 $V_{\mathrm{DWL}}$ ——在一批中具有最大平均直径的滚针和有最小平均直径的滚针其平均直径之差。  
(10)滚针公差等级——直径和形状公差的特定组合。用数字表示。
