# 发展概述

MC-H于2000年问世。为芬兰武装部队提供装备；2000年中期开始向皇家荷兰陆军供货两套。

# 技术简介

该通用、全封闭、新型活动弹射器能适用于各种飞行器（无人机或航空靶机），并能适应客户要求进行装备。斜轨长度可根据飞行器进行改变。弹射器可安装在卡车上、越野车上或坚实的基座上；卡车安装型，可用高达80km/h的速度运输，且其使用准备时间少于15min。

# BTW832绞盘

类型：拖靶绞盘

主承包商：Bulow靶机系统公司(BulowAirTargetSystemsAB)

# 发展概述

Bulow公司的第一种空气冲压涡轮为MBV-2S，其中的10个在1965年进入瑞典空军服役，其他10个为瑞士空军所用。BTW832是计算机控制的MBV-2S的发展型。

# 技术简介

安装在吊舱里的BTW832是一种能放出和缩回各种拖靶(机翼式、标枪式、袖套式、旗靶式)的通用绞盘。对袖套式拖靶，只有头锥或MDI被缩回，织品袋本身将被丢弃。

吊舱用NATO标准的 $356~\mathrm{mm}$ 的炸弹挂钩安装在军用飞机上；有专用的安装板以便于装在民用飞机上。有可向下倾斜的长发射臂，提供较低的牵引/缩回点，使拖靶免受牵引飞机的干扰。

绕进/绕出阶段由计算机自动控制，以便拖曳飞机为单座时，绞盘可由驾驶员控制。事实上，如果需要，无线电控制盒能使驾驶员同时操纵两个绞盘和两个靶标。绞盘吊舱带有内置发电机以自身供电。

# HTW 550 和 TTW 500/550 绞车

类型：直升机用(HTW550)和固定翼飞机用(TTW500/550)拖靶绞车

主承包商：Bulow靶机系统公司(BulowAirTargetSystemsAB)

# 发展概述

TTW550直升机拖靶绞车系为瑞典海军设计，用于拖曳大型海面掠海式拖靶。TTW550是用于固定翼飞机的改进型；TTW500除动力源外与之相近。

# 技术简介

HTW550绞车安装在GFRP的流线型盒内，并用 $12\mathrm{m}$ 长的粗缆绳吊挂在直升机的货钩上。绞车用座舱内的控制显示盒进行引导和监测，为此有一根穿过座舱门引出的细线与之相连。汽油发动机作为动力源通过液压马达驱动绕线鼓轮。为在任务前后能够停住，绞车被夹持在带有刹车齿轮的坚固框架内。尽管设计是专用于牵引掠海式靶标，HTW550是一个多用途卷线机，可用来拖曳各种航空靶标或雷达校准球，亦可用作直升机内的靶标高度显示和MDI接收站。

TTW550为一种改型，设计用于固定翼飞机，安装在附属于飞机座椅扶手的框架上。它由3个主要部分组成：绞车单元、液压单元和发射/拾取单元。前两者通常位于重心附近，而发射/拾取臂则位于恰在门内的座椅扶手上。该臂可用液压使其转动，向下为牵引，中间是为起飞和着陆，而全部向上是为更换MDI和袖套。缆绳速度的调节，通过手持操纵盒，用电子比例气流阀进行控制。

除液压系统为电驱动外，TTW500基本上和TTW550相类似。电流消耗为 $190\sim 240\mathrm{A}$ （峰值300A）、电压为 $24\sim$ $30\mathrm{V}_{\circ}$

TTW550和TTW550都用13.4kW(18hp)、四冲程气冷汽油发动机作动力；TTW500则用拖靶飞机的电源系统或单独的电池作动力

类型：液压式弹射器

主承包商：RUAG宇航公司/前瑞士飞机和系统公司(RUAG Aerospace, previously Swiss Aircraft and Systems Enterprise Corporation)

# 技术简介

RUAG宇航公司是为瑞士“巡逻兵”(Ranger)侦察型无人机和靶机研制的一种无人机发射系统。“巡逻兵”型发射系统紧凑地安装在越野式运载工具上，因而将机动性与自主性相结合。由于能在近1s的时间内释放高达 $600\mathrm{kW}$ 的功率，因而能以 $30~\mathrm{m / s}(108$ $\mathrm{km / h})$ 以上的发射速度，发射重达350kg的飞行器；飞行器发射的空中距离约为 $14\mathrm{m}_{\circ}$ 该系统便宜，易于使用和维修，安装迅速，且只需2个操作人员。

2001年，RUAG引进发展型名为Archer，为便于空运和不依赖用户的卡车队，设计成拖车型。除最大的无人机重量稍有下降外，其性能和“巡逻兵”型等同。

四段便携式轨道用夹板和销钉装配，并和以运载工具为基础的前面部分相连接；整个过程不需用工具。骨架型面为三角形，用不锈钢管焊接而成。即使在全部伸长的情况下，斜轨也易于移动到合适的发射方位。

飞行器的发射是靠滚轮上的低惯性小车，加速是靠一根由合成材料制成的皮带提供峰值为 $25\mathrm{kN}$ 的拉力。由两个液压马达驱动的皮带缠绕器安装在紧靠最大发射速度点的下方。当小车在皮带缠绕器上方达到该点时，缠绕器反转其转动方向，因而减速并将飞机释放出去。空小车被反向驱动马达迅速停住，作为顶住高压气体的液压泵。

发射能量由充有压缩氮气的活塞

蓄压器提供。随温度而变的气体压力(最大350bar)决定弹射功率；气体永久储存在封闭的系统里。活塞储压器可用小型泵以大约 $12\mathrm{min}$ 时间重新充压，该小型泵或者用运载工具的电池或者用外部电源带动。以微处理机为基础的控制系统用来检测各种参数，如滑油压力和容积、驱动电子阀门和控制使用安全。

用RUAG弹射器发射“巡逻兵”无人机的照片见本章首页。

# Frazer-Nash 大气压弹射器

类型：大气压弹射器

主承包商：Frazer-Nash 防务系统公司(Frazer-Nash Defence Systems Ltd)

# 技术简介

这种非常简单的弹射器基本上是由一个轻质汽缸组成，汽缸内有一驱动活塞和一对置的活塞，发射轨道就安装在汽缸上。驱动活塞通过缆绳和滑轮系统直接与飞行器发射小车相连，用以使小车沿发射轨道加速，而对置活塞则用来在汽缸内产生真空。据说这种概念比橡皮筋和气动弹射器有很多优点，可用于发射重量为15~200kg的飞行器。

操作程序是先把发射小车返回到

出发位置，并把退缩锁锁上。此时两个活塞被拉到汽缸的前端。驱动活塞用驱动绳牵引，对置活塞则由于这种运动造成的吸力而跟随运动。无人机装在发射小车上，起动发动机并进行飞行前检查。

在即将发射前，对置活塞被电绞车拉向汽缸后方，从而在两个活塞间造成真空。连接驱动活塞和发射小车的缆绳穿过固定在轨道上的滑轮，该滑轮的位置则在飞机腾空点附近。打开退缩锁，大气压就会使驱动活塞加

速。该运动通过缆绳和滑轮传递给发射小车并使无人机沿发射轨道加速。当发射小车通过缆绳前滑轮时，驱动缆绳和驱动活塞将阻止继续运动，此时，同样是利用大气压，两者的作用是刹车。

设计特点为紧凑(发射滑轨短)；操作简单；半熟练人员即可操作；最大便携式构型；无需后勤支持；寿命周期成本低。

用于南非ATE“秃鹫"(Vulture)系统。

# HP 液压气动弹射器

类型：液压气动弹射器

主承包商：工程系统公司(Engineering Systems,ESCO)

# 发展概述

在研制用于战斗机和民用机、航天飞机以及各种无人飞行器的发射和/或回收设备方面，ESCO公司已有30年以上的经验。用户包括飞行加油公司和发展科学公司。

# 技术简介

使用气体加压液压蓄气瓶的无人机液压气动(HP)弹射器系列，代表小型飞行器加速器技术的当前水平。弹射时，蓄压瓶内储存的能量通过驱动汽缸和绑住的十字头把无人机加速到弹射速度(最大可达 $10\mathrm{g}$ )。这种设计导致峰值到均值的效率超过 $95\%$ ，且使无人机有接近常数的加速度。该HP弹射器系列已通过军用环境试验且能在极端温度、湿度和其他气候条件下使用。弹射时压缩气体并不消耗。

HP弹射器按定做的方式供货，大小和内容都可按用户对性能和自动化水平的要求制作。这种弹射器有内锁安全子系统，和确定的释放机构以防

止偶然弹射。滑梭刹车的确认是用旋转的液压涡轮。操作和维修不需特别的技术和工具。任选内容有固定式或活动式安装；汽油，柴油或电源动力装置(名义要求 $18.6\mathrm{kW};25\mathrm{hp}$ ）；绞接的

弹射轨道；滑梭接口按真实的无人机机构型定做；还有很多自动化子系统。ESCO的HP弹射器所覆盖的无人机加速度范围为 $10\mathrm{g}$ 以下。弹射器已安装在各种军用和民用卡车和拖车上。

# ESCO 回收系统

类型：降落伞和网撞回收系统

主承包商：工程系统公司(Engineering Systems,ESCO)

# 技术简介

ESCO研制出一种垂直网和水平网组合的无人机回收系统。这种系统储存时很紧凑，而且几分钟即可投入使用。它需要很少点或不需要固定，且能在很大范围内适用于无人机的重量和进场速度。使用这种系统，可不考虑风向和风速，地面情况(岩石等)亦不必考虑，且回收地点可加以确定并保持为最小。在1990年初美国陆军Yu-ma试验场举行的近程无人机试验时，该系统和ESCO弹射器一起成功地进行了验证。

另外一种网撞系统是ESCO的

SPARS("先锋"无人机船上拦阻系统)，是设计和安装在船上用于美国海军的“先锋”无人机。该系统有垂直网，并在1991年海湾战争中得到成功的应用。

MARS(中空回收系统)已成功用于降落伞悬挂无人机的中空回收近30年。ESCO设计和研制的这种系统既可用于直升机也可用于固定翼，已实现20000架以上无人机的回收，重量 $91\sim 1814\mathrm{kg}$ 。MARS系统使用了ESCO80H型张力控制，能量吸收绞盘，可使该系统适用于大范围的直升机和固定翼飞机。

# LTC-2和LTC-6弹射器

类型：拖靶弹射器

主承包商：梅吉特防务系统公司(Meggitt Defence Systems, formerly Hayes Targets)

# 发展概述

这种弹射器设计用于MeggiHUTTS系列靶机，而且可用于任何能携带外挂的飞机。对装有 $356~\mathrm{mm}$ 或 $762~\mathrm{mm}$ 标准炸弹挂钩的各种飞机，不需任何改装就可安装和使用。其轻型的气动力设计只对飞机性能产生很小的影响。1970年以来，一直在生产和使用。

# 技术简介

LTC-2基本型弹射器由吊舱、悬挂释放器、作动器和头锥组成。吊舱有两个挂钩和飞机的外挂舱相连接。悬挂释放器连在吊舱上用于悬挂靶机和连接牵引绳。对Meggitt只弹射不回收类型的靶机，LTC-2使用内部牵引绳。用于靶机弹射和释放牵引绳的作动器，或者做成机电机构或者用烟火袋作动器操作。使用TIX红外拖靶时，这种构型还外加点火装置。

LTC-6型的牵引长度可更长，它由LTC-2基本型和在吊舱后部加装的卷筒刹车装置以及绳切割器组合而成。这些附加装置形成牵引绳卷筒的外罩并对其进行控制，牵引绳卷筒则和靶机上的牵引绳相连。绳切割器用烟火作动以释放LTC-6的牵引绳。

两种弹射器可以是硬线式，或加进遥控装置进行遥控，以及使用驾驶员或机组成员的手持发射机。对用缠

绕机的往返式系统，装有备份设备，以避免缠绕机故障时任务时间损失和财产损失。

# 技术简介

RM系列缠绕机是驾驶员操纵的双向缠绕系统，它能回收拖靶并再次使用。各型号都有自动缠绕功能(放出、停止和收进)，装有双重/余度牵引绳切割器，有故障-安全刹车系统，且可中心安装或侧面安装。其他特点随型号而不同。

# TALS 着陆系统

类型：战术自动着陆系统  
主承包商：Sierra Nevada公司

# 发展概述

Sierra Nevada 战术自动着陆系统 (TALS) 是该公司的无人机共用自动回收系统 (UCARS) 的缩小型。AAI 公司的 RQ-7A “影子” (Shadow) 200 战术无人机系统有订货。2000 年美国陆军进行过试验。

# 技术简介

TALS是K波段(35GHz)、全天候应答式跟踪雷达系统，设计用于陆基环境和与任何地面控制站(GCS)交互工作。它是一种毫米波雷达，有波束宽度窄的天线，近距使用于雾、雨、亮光或夜间；拦截的可能性低；对主机影响最小；大量的自检测设备便于维修；准备时间少于 $15\mathrm{min}$

这一重量很轻的机载应答器用于精确跟踪的点信号源，定向天线用于获取，而全向天线用于滑行。包含在便携式单元里的地面跟踪分系统对机载应答器进行定位并精确跟踪，用高带宽的跟踪回路覆盖接地和滑行。U-CARS中经过考验的回收软件完成飞行器的导航和控制计算。系统自备的发射和回收数据链可作为自选项提供。

# UCARS 回收系统

类型：无人机共用自动回收系统主承包商：Sierra Nevada公司

# 发展概述

1990年，美国联合计划局(JPO)和Sierra Nevada公司签订研究合同，开始无人机共用自动回收系统(UCARS)的研究，目标是开发一种能用于回收当时JPO无人机系列中任何一种无

人机的回收系统。从那时起，就开始了原型机的研制、集成和验证计划，其基础是使用K波段(35GHz)毫米波雷达。其他技术也评估过，但毫米波仍是最有前途的。

用庞巴迪CL-227为飞行器.按

MAVUSI计划于1992年7月在奥可拉荷马州进行过8次自动着陆，每次回收都在预定接地点的 $0.25\mathrm{m}$ 范围内。按MAVUSⅡ计划(1993年12月到1994年1月)，把自动着陆的计算程序改进之后，26次着陆当中都位于接地

点的 $0.5\mathrm{m}$ 范围内。随后1994年2月，在USSVandegrift(FFG-48)甲板上进行了海上试验。1996年12月，“先锋”无人机成功地对它进行了验证；其后1997年初在USSShreveport两栖登陆舰甲板上亦进行过试验。

# 技术简介

UCARS是一种毫米波跟踪雷达和导航系统，向飞行器提供位置数据和导航指令，以在好天气或坏天气条件下，以及陆上或海上自动回收无人机。其设计是能与任何一种地面站(GCS)，以及与固定翼，旋转翼，倾斜翼，以及倾斜旋转翼式无人机共同工作。系统是便携式，空中机动式和自校准式；其他使用情况包括特殊应用和秘密使命。

UCARS由3个基本子系统组成：

机载子系统：波束模式时，发出脉冲使跟踪子系统开始跟踪；应答模式时，回应跟踪子系统的问询以计算距离。一个小型、波束很窄的天线可按使用的无人机的特殊要求进行剪裁。

跟踪子系统：发射问询脉冲以在方位、俯仰和距离上跟踪机载子系统；计算飞行器的位置数据（如果装在船上，则用船的运动传感器进行稳定）并把位置和运动传感器的数据送往回收子系统。上行数据已用19000波特率进行过验证。

回收子系统：这是由操纵手确定的有关飞行器类型和进场的剖面信息。它显示飞行器的位置和状态，计算飞行器的飞行指令，并通过地面站的数据链把这些指令发送给飞行器。
