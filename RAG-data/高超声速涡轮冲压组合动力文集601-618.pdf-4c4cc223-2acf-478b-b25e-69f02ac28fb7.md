[摘要] 在Kakuda研究中心的国家宇航实验室，利用冲压发动机试验台，在来流马赫数为5的自由射流条件下，对高超声速运输机冲压发动机的缩尺模型样机进行了研制和测试。在最近的测试中，对新设计的低外阻模型样机进行了测试，与此同时，也验证了进气道端部的激波控制技术。测定超音进气道端部激波的位置，并且将其和燃油流量同时进行控制。分区供油试验结果表明，如果激波位置波动和进气道压力增加，会导致发动机熄火裕度的缩小。因此建议，要随时监测激波位置波动和进气道压力，以避免熄火。如果释放空气量由 $1\%$ 增加到 $12\%$ ，模型发动机测得的净推力会减少 $46\%$ 。实验净推力和计算净推力相差大约 $7\%$ 。

# 1 引言

Hypr 计划中已经研究和发展了马赫数 5 一级的高超声速运输机的推进系统。在评估中选择了串联式的联合循环发动机，联合循环发动机由一台可变涵道比的小涵道涡轮风扇发动机和一台冲压发动机组成。图 1 是这项研究中的备选方案之一，发动机是安装在机身或机翼下面。冲压发动机使飞机由马赫数 2.5 加速到 5，并且在马赫数 5 的速度下进行巡航。

冲压发动机系统的研究和验证发动机的发展同时进行。在固定几何和纯冲压模态的条件下，对缩尺模型发动机进行了测试。在第一次试验中，证实了稳态燃烧可以达到计划目标温度 $1900^{\circ}\mathrm{C}$ ，同时冲压系统工作高效而且稳定。实验虽然是成功的，但是也存在两个问题。一个问题是，由于外部阻力过大，测的推力是负值。另一个问题是，采用预调制型的开关燃料阀，进口的临界工作状态很难实现。为了比较容易的实现接近临界的工作状态，避免进口失效，需要探测和控制进口激波位置。在第二次试验中，试验研究的重点主要集中在减阻和激波位置控制方面，第二次试验的目标分别是：

(1)燃料流量与超音进口的激波位置协同控制；  
(2) 减少发动机的外部阻力。

为了减少外部阻力，改变进口形状，减少发动机前端面积。同时使用了在激波下游直连实验中获得的激波位置探测和控制技术。由于是固定几何的发动机，因此，采用可变燃

料阀来控制激波的位置。

# 1.1 试验装置和冲压发动机

冲压发动机试验装置安放在一个长6米，直径3米的测试间里。冲压发动机试验装置的喷管出口尺寸是，高 $510\mathrm{mm}$ ，宽 $510\mathrm{mm}$ 。冲压发动机的尺寸据此决定，并且完全暴露于空气射流之中。冲压发动机由浮动框架测力系统(FMS)支撑在自由射流来流中。FMS配有机械锁定系统，用来避免风洞启动和关闭时造成的突然破坏。在试验之前，对自由射流条件进行校准。试验条件是： $\mathrm{Ma} = 5.5$ ，总压 $= 3.5\mathrm{MPa}$ ，总温 $= 600\mathrm{K}$ 。运行时间60秒。冲压发动机依照此流动条件进行设计。图2为用于第二次自由射流试验的冲压发动机。

# 1.2 进气道

在冲压发动机系统的研究过程中，为了产生足够抵抗大约是马赫数5飞行条件下总压3倍的自由射流总压的力，采用了固定几何的自启动型的进口设计方法。进口收集面积根据 $100\mathrm{mm}$ 高 $\times 100\mathrm{mm}$ 宽的来流面积决定(包括放气和溢出)。图3为第二次试验的进口剖视图。进口通过8个斜坡和一个罩状内壁压缩超音空气。一个亚音扩压器与冲压燃烧室相连，它的截面形状由矩形逐渐变为圆形。在第8级斜坡、罩状内壁和喉道上有放气孔或长缝，用来去除边界层和端面激波。为了观察流动情况，在接近喉道区域的壁面上设置了观察窗。侧壁可以防止第三级斜坡外部压缩区域的侧面溢出。为了探测激波位置，在6个测试装置上都设置了压力测孔，如图3所示。压力测孔的间隔为 $10\mathrm{mm}$ 等间距。最上游的测量孔被放置在进口喉道下游4mm处。

在第一次设计中，优先考虑启动特性，而且外部压缩转角被设置为40度，接近主流的最大转角。内部压缩的收缩率是0.69，这个数值是对应于内部压缩区域进口马赫数的K-D界限的 $90\%$ 。结果，罩状内壁的阻力增加。

在第二次设计中，为了减少罩状内壁的阻力，将外部压缩角改为30度。虽然减少了进口的外部阻力，但同时也降低了压力恢复系数。为了弥补压力损失，斜坡激波数由4增加到8。收缩率稳定在K-D限制的 $82\%$ 。两次进口试验的不同由表1给出。

针对进口和超声速风洞的起动特性，进行了许多试验研究。Van Wie 等人将启动界限的试验数据与 Kantrowitz 和 Donaldson 基于一维分析得到的起动界限进行对比。

K-D界限并不是一个必要条件，而是一个起动的充分条件。例如，他们所研究的在马赫数3条件下超音进口的起动和再起动特性，允许的起动和最大收缩比分散于K-D界限和相应的等熵界限之间。二维欧拉分析表明，第二次设计进口的气流由于进口斜坡的外部压缩减速到马赫数3.1，然后进入内部通道。因此，基于K-D界限的允许收缩率是0.72。另一方面，等熵压缩的收缩率是0.59。因此，第二次设计进口的临界（Swallowing）函数值是 $0.82(= 0.59 / 0.72)$ 。也就是，设计的内部通道比K-D界限决定的内部通道更小。相应于临界(Swallowing)函数值0.82的收缩率在K-D界限和等熵界限之间。在临界(Swallowing)函数值是0.82时，由于多孔板放气(多孔板被放置在第8级斜坡和罩状内壁的放气孔处，以及喉道后的长缝处)，自起动是可以实现的。

# 1.3冲压燃烧室

# 2 试验结果和讨论

# 2.1 进气道特性

为了获得进口流量和压力恢复系数之间的关系，在发动机试验之前，先对进气道单独进行试验。进气道的背压由进气道末端的阀门来调节。图7表示在没有放气条件下的进气道特性。纵轴表示压力恢复系数，横轴表示质量流量。

质量流比率定义为，理想质量流量与进气道完全收集的自由来流流量的百分比。最大压力恢复系数是 $22\%$ ，最大质量流量是 $82\%$ ，如图所示。和第一次的进气道自由射流试验相比，最大压力恢复系数降低了，最大质量流比率增加了。进气道的几何收集面积比第一次试验时减少了 $24\%$ 。最大质量流量是 $1.32\mathrm{Kg / s}$ ，比第一次试验时减少了 $8.5\%$ 。

# 2.2 进气道喉道附近的流动显示

图8是喉道放气孔附近在无风条件下的纹影仪照片。图中心的矩形部分是通过进气道侧壁观察窗拍摄的可视化图片。气流流向是从左至右。上部右侧至左端的楔形阴影是罩状壁。图片左下方是斜坡表面。在中部，有一个喉道放气室，空气流通过喉道槽缝隙被释放。在通道中的L形阴影是皮托管，用来测量喉部总压。在罩状壁上也有许多放气孔，它们的

位置与喉道放气孔相对应。在斜坡表面也设置了许多放气孔。纹影的颜色在空气被压缩的地方调整为红色，在空气膨胀的地方调整为蓝色，在无风的条件下根据空气的密度梯度将颜色调整为绿色。

图9显示的是斜坡放气阀关闭时的流动图像。边界层沿着斜坡表面发展。如果斜坡表面不放气，边界层的作用就相当于斜坡角增加，从而使斜激波角变大。由于斜激波角增加，在罩状壁边缘会形成分离激波。

相反，当斜坡表面放气时，在图10中的流动图像发生了很大的变化。在这种情况下，由于斜坡放气去除了边界层，斜坡表面的斜激波进入罩状壁内侧。形成了附着在罩状壁内侧的斜激波系。由于斜坡表面放气，在斜坡表面产生了膨胀波和压缩波。在皮托管左侧观察到的三条红线，是罩状壁放气孔形成的压缩波。在喉道放气室槽缝的背面形成了朝向下游的射流，显示为放气室内的小红斑。

图9和图10中平行于罩状壁下表面的黑色阴影是一个剪切层。在剪切层和罩状壁下表面之间的气流流速较低，因为这股流动正好位于一个强激波的下游，这股强激波是由罩状壁边缘和斜坡表面的激波系相交形成的。另一方面，位于剪切层内下侧的一股流动流速很高，因为它只通过了一个斜坡激波。结果，明显的速度不连续性和剪切层出现了。

# 2.3 发动机再起动

发动机熄火是通过激波位置的变化和重新点火的再起动程序来探测的。图11显示了测量推力和激波位置随时间的变化规律。由于从15秒开始，燃油流量稳定增加，在总当量比为0.83，23秒的时候发动机熄火。熄火之后，估计的激波位置是 $40\mathrm{mm}$ 。图中的激波位置跳跃反映了发动机熄火。在探测到熄火之后，马上把燃油流量从总当量比0.83调到0.62。发动机在熄火3秒后，再点火成功。

进气道很容易重新启动。即使发动机富油熄火，在停止供油后进气道会重新工作。Van Wie 等研究者把进气道失效分为两大类，一种是当进气道喉道阻塞时的“硬失效”和由于当地分离引起的“软失效”。他们也认为，K-D界限对于“硬失效”和再起动情况都是适用的。

在第二次自由射流试验中，从安装在进气道超音和亚音区域的41个测量孔获得的压力数据来看，没有观察到任何“硬失效”的证据。起动特性，既不依赖斜坡放气(质量流比率 $0\sim 8\%$ ），也不依赖放气(最大 $2\%$ 的质量流比率)，即使在试验中两个放气阀一直关闭，也可以实现自起动。

# 2.4 激波位置和推力

图12显示了，在再起动试验中测量推力、激波位置与总当量比的关系。横轴表示总当量比，纵轴表示测量推力和激波位置。当测量推力由400N增加到500N时，激波位置仅变化了 $5\mathrm{mm}$ 。当总当量比增加时，激波位置的离散点增加。

在测量推力大于400N的区域，激波位置是负值。这说明，端部激波的前缘已经到达喉部附近(激波位置的0点是在进气道喉部下游4mm处)。激波位置的误差估计并不精确，因为有数据已经小于-4mm，那意味着喉道上游的位置。就控制目的来说这个激波位置是有利的，因为它对激波位置的相对变化很敏感。发动机是固定几何型的，燃油流量增加时，进气道出口的压力上升会使激波的位置向上游移动。

# 2.5 激波位置控制

发动机是固定几何型的，因此控制激波位置的独立变量限于燃油流量(当量比)和放气流量。当改变斜坡放气量时，激波位置和燃油流量一起保持不变。图13显示了激波位置、测量推力、压力恢复系数，燃油流量，斜坡放气流量随时间的变化规律。在这次试验中，激波位置保持常数 $17\mathrm{mm}$ 。在第43秒的时候，斜坡放气阀由全关状态开到全开状态，以使空气流量减少 $8\%$ ，激波位置移向下游。与此同时，测量推力和压力恢复系数减小。然后，通过反馈控制器，将燃油流量增加，以便使激波位置回到初始位置。激波在第47秒的时候回到初始位置。而且，测量推力和压力恢复系数也恢复到原来的值。不管是对于避免熄火还是推力控制，改变燃油流量和改变喷管喉道面积一样，都是保持激波位置为常数的好方法。

# 2.6 燃油分区

进行燃油分区试验是为了研究空间燃油分布对燃烧，发动机稳定性和进气道激波的不稳定性的影响。图14显示了总当量比、主2区供油相对总供油量的比率和激波位置随时间的变化规律。在本次试验中经历了两次熄火。第一次熄火是在激波位置-6mm，主2区燃油比率0.19，总当量比0.82时发生的。另一次熄火是在波位置-2mm，主2区燃油比率0.61，总当量比0.65时发生的。这台发动机，当主2区的供油比率增加时，熄火会发生在更下游的激波位置和更小的总当量比条件下。注意激波位置随时间变化的规律，可以看出，二次熄火的激波位置比第一次熄火的激波位置更加分散。

图15显示了进气道压力恢复系数随时间的变化情况。第二次熄火发生在压力恢复系数在0.20时，而第一次熄火发生在压力恢复系数在0.21时。当主2区的供油比率增加时，熄火时的平均压力恢复系数减小。因此，为了避免熄火，有必要在下游设置目标激波位置或者不增加主2区的燃油比率。燃油分区可能会引起激波位置和进气道压力波动的增加，缩小发动机熄火边界，如图14和图15。为了很好的避免熄火的发生，需要在短时间内感受到激波位置波动和进气道内压波动的平均值。

# 2.7 发动机推力

发动机的测量推力 $F_t$ 由测力系统获得。图16显示了总当量比和测量推力之间的关系。横轴代表总当量比，纵轴代表测量推力。进气道放气比率为 $1\%$ 时用符号 $\bullet$ 表示， $12\%$ 时用 $\triangle$ 表示。进气道放气气流并不是直接排向自由射流中，而是通过管道引入低压室。因此，当进气道放气使得空气流量减小时，喷管出口的气流动量通量也减小。例如，在接近临界工作状态时，放气量从 $1\%$ 增加到 $12\%$ 时，发动机测量推力减少 $45\%$ ，如图所示。

为了与计算的净推力相比较，必须估计基于测量推力的净推力。基于测量推力的净推力由下式给出：

$$
\mathrm {F n} = \mathrm {F t} + \mathrm {D e x t} \tag {1}
$$

对外部阻力Dext进行分步估算。它包括斜坡下边阻力，侧壁阻力，罩状壁阻力，溢出附加阻力，发动机座阻力和摩擦阻力。在这些阻力中，斜坡下边阻力、侧壁阻力、罩状壁阻力和发动机座阻力都是它们自身各部分的压力阻力，可以通过测量发动机各部分的外壁压力来计算。溢出附加阻力和摩擦阻力也可以通过测量数据和计算数据估算出来。附加阻力作用在外部溢出流和进气道内部流动之间的边界上。虽然附加阻力通过边界上的压力-面积积分进行计算，但是在试验中边界并不是可以观察的。因此必须通过进气道的性能数据估算边界面积dA。假定边界上的平均压力Pa等于经过总转角的2/3角度压缩后的气流

压力。不同于压力-面积积分方法，溢出附加阻力可以通过下面的式子计算，

$$
\mathrm {D a} = \mathrm {P a} \times \mathrm {d A} \tag {2}
$$

摩擦阻力可以通过积分所有作用在外表面上的粘性力得到。粘性力的估算是基于压力和马赫数的，这些值是由每一个位置的Euler分析和平均壁面摩擦系数获得的。平均壁面摩擦系数由Frankl-Voishel公式计算。表2列出了由上述方法得到的发动机外部阻力估算。外部压力阻力的计算是由测量的绝对压力得到的。因此，发动机座阻力的结果表达为负值。

由于斜坡激波的形状变化(如图9和图10)，当斜坡放气量增加时，进气道溢出减少。因此，斜坡放气增加时，附加阻力减小。根据表2可以看出，溢出附加阻力不到总阻力的 $10\%$ 。罩状壁阻力最大，占发动机外部阻力的 $46\%$ 。正如上面提到的，净推力由发动机试验数据进行估算。另一方面，净推力也可以通过一维流动方程和部件试验数据获得。

根据部件试验结果，在性能计算中，使用了图7中的质量流比率和压力恢复系数， $94\%$ 的燃烧效率，0.922的喷管出口速度系数。因此，在计算中的内部摩擦损失通过部件试验结果来考虑。部件试验获得的燃烧效率与第二次自由射流试验的气体取样分析结果符合很好。在计算中，并没有考虑未燃气体的质量流对推力的影响。

图17比较了由测量数据获得的净推力(试验值)与性能计算获得的净推力(计算值)。横轴表示总当量比，纵轴表示净推力。符号 $\bullet$ 表示 $1\%$ 放气流量时的净推力，符号▲表示 $12\%$ 放气流量时的净推力。实线表示净推力的计算结果。在 $1\%$ 放气量的条件下，计算值与试验值相差在80N以内。在接近临界的工作条件下，最大的差值是计算净推力1132N的 $7\%$ 。在 $12\%$ 的放气条件下，两者差别很小。

# 3结论

(1) 超声速进气道端部激波位置的探测和控制与燃油流量的调节同时进行。  
(2) 燃油分区可能会引起激波位置波动和进气道压力波动的增加, 缩小发动机熄火边界。为了避免熄火的发生, 应该探测进气道内的激波位置和压力波动。  
(3) 在接近临界的工作状态下，将放气量由 $1\%$ 增加到 $12\%$ ，模型发动机的净推力会减少 $45\%$ 。  
(4) 试验净推力与计算净推力相差 $7\%$ 。
