# 3 使用工艺

3.1 成品件外表面可用气相防锈纸覆盖或包裹，对成品件内腔可将气相防锈纸卷成筒状，涂防锈剂面朝外，塞入内腔中。气相防锈纸与被防护金属表面的距离不得大于 $30\mathrm{cm}$ 。注意与非金属材料的适应性（参见亚硝酸二环己胺）。  
3.2 由于气相缓蚀剂对手汗没有置换性能，所以产品在封存前应认真清洗。可选用添加 $3\% \sim 5\%$ 置换型防锈油的汽油或煤油清洗或用含水溶性缓蚀剂的水基清洗剂清洗。  
3.3 注意气相缓蚀剂的有效使用半径和死角。亚硝酸二环己胺的有效使用半径为 $30\mathrm{cm}$ ，碳酸环己胺的有效使用半径为 $60\mathrm{cm}$ 。当超过有效使用半径时，会失去对金属的保护作用。当被包装物的某些内腔气相缓蚀剂的作用达不到时，便成了死角，应在死角处撒上气相缓蚀剂粉末或放上一些气相防锈纸。  
3.4 要解决包装空间中气相缓蚀剂达到有效保护浓度前这一段时间内易锈金属可能产生锈蚀的问题，应在清洗用的汽油或煤油中加入 $3\% \sim 5\%$ 置换型防锈油或脱水防锈油，使零件表面有一层很薄的防锈油膜，或在清洗的水剂中加入水溶性缓蚀剂等，以防止气相缓蚀剂尚未充分发挥作用之前的金属锈蚀。  
3.5 在封存期间，包装内的相对湿度不得超过 $85\%$   
3.6 气相缓蚀剂在较高的温度和紫外线直接照射下会发生分解变质失去缓蚀效果。因此，使用时温度不宜超过 $60\mathrm{C}$ ，并且避免光线直接照射。  
3.7避免与酸、碱性物质接触。微量的无机酸能使亚硝酸铵变成亚硝基胺而失去缓蚀效果。过量的碱则会形成游离胺和碱金属的亚硝酸盐。因此，实际使用时，必须避免包装纸、木箱、缓冲材料等挥发出的酸碱性物质与气相缓蚀剂接触。

# 2.3 耐环境性能

2.3.1 耐酸、碱、盐性能 聚乙烯薄膜具有优良的耐腐蚀性能，优异的抗水性和耐酸、碱、盐类水溶液的腐蚀，在 $60^{\circ} \mathrm{C}$ 以下耐 $50\%$ 的硫酸、 $40\%$ 硝酸、浓盐酸和各种浓度的盐、碱类溶液的腐蚀。对氢氟酸、甲酸的耐腐蚀性也很好，但它不耐强氧化性酸的腐蚀。  
2.3.2 耐溶剂性能[4] 聚乙烯薄膜在室温下几乎不溶解于任何有机溶剂，但脂肪烃、芳香烃、卤代烃类能使其溶胀，温度超过 $70^{\circ} \mathrm{C}$ 时，聚乙烯可以少量溶解在苯、甲苯、乙酸戊酯、三氯乙烯、松节油和氯代烃中。密度不同的聚乙烯耐溶剂性也不一样，高密度聚乙烯结晶度高，耐溶剂性相对好些。聚乙烯薄膜的分子量对耐溶剂性也有一定的影响，低分子量部分较易溶解，高分子量部分则要在较高温度方可溶解。  
2.4 工艺性能 聚乙烯薄膜的色泽呈乳白色，膜面上无穿孔和粗糙条纹，允许存在 $0.3 \mathrm{~mm}$ 以下极小杂质点。

# 2.3 耐环境性能

# 2-3.1 耐腐蚀性能

（1）聚氯乙烯薄膜对水、盐溶液相当稳定，但对氧化性盐溶液的抵抗能力随浓度的增加而减弱。如在稀的次氯酸钠溶液中稳定，但在浓的次氯酸钠溶液中在有应力存在下易发生破坏。  
（2）聚氯乙烯薄膜对大部分酸介质稳定，但少数氧化性酸浓度较高时有较大的腐蚀作用。它可在质量分数 $50\%$ 以下的硝酸、 $80\%$ 以下硫酸中使用。浓度再高的硝酸就易使聚氯乙烯薄膜变白、起泡、开裂。质量分数 $90\%$ 以上硫酸易使聚氯乙烯薄膜表面变黑。浓盐酸能使聚氯乙烯薄膜变紫，但对性能影响不大。聚氯乙烯薄膜对氢氟酸的耐腐蚀性较好，但渗透能力较大。聚氯乙烯薄膜有较好的耐碱性，但高温浓碱能促使聚氯乙烯薄膜脱氯化氢反应的进行，从而降低耐蚀性能。  
（3）聚氯乙烯薄膜耐丙烷、丁烷、石油、汽油、白油和煤油等的腐蚀，也耐脂肪酸如甲酸、乙酸和草酸，醇类如乙二醇、丙三醇和乙醇等的腐蚀。但芳烃如苯、甲苯、氯苯、氯代烷烃，酮和醚这些中等极性的溶剂易使其溶胀、分层。有些有机介质会与聚氯乙烯薄膜起反应而使其破坏，如与苯胺的反应在高温下进行得较彻底，常温下长时间也会破坏。在室温下能溶解聚氯乙烯薄膜的溶剂有环己酮、四氢呋喃、硝基烷和吡啶等。

# 2.3.2 耐热性能

聚氯乙烯薄膜的耐热温度在玻璃化温度附近，它在 $80\sim 85^{\circ}C$ 开始软化， $160^{\circ}C$ 开始呈流动状态。聚氯乙烯薄膜中存在不稳定基团，如仲、叔氯、端基中的催化剂残基、不饱和端基、头-头、尾-尾结构等，因此聚氯乙烯薄膜的热稳定性很差，在 $100^{\circ}C$ 以上就开始有HCl逸出， $140\sim 180^{\circ}C$ 分解作用较显著，形成共轭双键，超过 $200^{\circ}C$ 开始激烈分解。聚氯乙烯薄膜耐燃性较好，但在火焰中会燃烧，放出刺激性的氯化氢气体，一离开火焰即刻自熄。

# 2.4 工艺性能

2.4.1 软聚氯乙烯吹塑薄膜应塑化良好，不允许有气泡、穿孔、破裂，允许有 $0.3 \mathrm{~mm}$ 以下的杂质存在；不允许有 $0.6 \mathrm{~mm}$ 以上的杂质存在。 $0.3 \sim 0.6 \mathrm{~mm}$ 的杂质许可量应小于 $10 \text{个} / \mathrm{m}^{2}$ ，分散度应小于 $3 \text{个} / 5^{2} \pi \mathrm{cm}^{2}$ 。允许有 $0.3 \mathrm{~mm}$ 以下的晶点存在， $0.3 \sim 1 \mathrm{~nm}$ 的晶点其分散度为 $15 \text{个} / 5^{2} \pi \mathrm{cm}^{2}$ 。不允许有分解线存在，允许有轻微的水纹及符合公差的厚薄道。薄膜表面及卷曲应平整，边缘处允许有少量的活褶。

# 硅胶

# 1 概述

硅胶是航空零部件及产品包装用的干燥剂。是一种硅酸溶液凝结而成的人造含水硅石。具有高微孔结构、较高的热稳定性和化学稳定性，质地坚硬、无毒、无味、不燃、不爆、无腐蚀性、亲水性强，但不溶于水等特点。由于其制造方法不同，所以有不同的微孔结构和比表面积。细孔球形硅胶主要用于做气体净化剂、干燥剂、空气湿度调节剂和催化剂载体，并用于石油产品的精制；细孔块状硅胶主要用做气体净化剂、空气湿度调节剂，在冷冻过程中作吸附剂，并用于石油化工产品的精制分离和有机产品的脱水精制；粗孔球形硅胶主要用做气体净化剂、干燥剂，以及绝缘油的除酸剂；粗孔块状硅胶主要用做催化剂载体、干燥剂、气体和液体净化剂；蓝胶指示剂主要用于指示干燥剂的饱和程度及密封包装物、精密仪器仪

表内空间介质的相对湿度；变色硅胶用做干燥剂，主要用于一般密封包装物和仪器仪表的防潮。

1.1 牌号及名称 细孔球形硅胶、细孔块状硅胶、粗孔球形硅胶、粗孔块状硅胶、蓝胶指示剂和变色硅胶。

# 3 使用工艺

3.1 干燥剂在使用前的含水率应不大于 $4\%$ ，否则应予以干燥后备用。  
3.2 盛装干燥剂的容器应能易于吸收防潮包装内的湿气，而其粉尘又不会落在内包装物上。为此将其分装在透气性良好的细布袋、纤维纸(无纺布)袋或有小孔的金属罐内。为便于使用，可将干燥剂预先分装成每袋一定量值（或常用量值），并按质量分罐盛装。  
3.3 包装时，干燥剂在空气中暴露的时间，最长不得超过 $10\mathrm{min}$ 。当由于偶然性或其他原因而时间较长时，则应烘干后方可重新使用。  
3.4 各种硅胶再生时的实际烘干温度和干燥时间见表3-1。烘干再生时，应注意使之逐步提高温度。烘干时厚度不应大于 $30\mathrm{mm}$ ，每小时至少搅拌一次。在烘干过程中，应注意使硅胶或变色硅胶等不与油或其他挥发性物质混在一起，以避免硅胶吸附这些物质后降低使用效率。烘干结束后，应冷却至 $60\sim 70^{\circ}C$ ，立即使之贮存于密封的容器中备用。

3.5 干燥剂袋或罐应均匀地分布于防潮包装内，使包装内所有空间都能受到干燥剂的去湿作用。或是放在被包装物需要干燥的部位或其附近，但不得直接搁置在包装产品上，不得与金属或油漆表面直接接触。干燥剂的袋子如接触涂覆防锈油脂的表面时，必须用无腐蚀耐油性包装材料隔离。

3.7 为防止干燥剂袋或罐在贮运中摆动或摩擦破损，应在包装时将其加以固定。如用细绳将其系住或锁定。  
3.8 为了便于了解包装内部的相对湿度情况，采用湿度指示剂、湿度指示纸或湿度指示装置。湿度指示剂（纸）和湿度指示装置的位置应尽量远离干燥剂。  
3.9 粗孔型硅胶不宜用来作为封存包装用干燥剂。

# 蒙脱石

# 1 概述

蒙脱石是航空零部件及产品包装用的干燥剂。是一种结晶层状铝硅酸盐粘土，其晶体构造是由两层硅氧四面体和一层铝氧八面体构成，在两层硅氧四面体中间充满着吸附水和可交换的阳离子。由于离子交换的结果，晶体表面带负电荷，钾、钠、镁、钙、铁、铝等离子用于电荷平衡，水分子易于进入晶层之间发生膨胀。蒙脱石的吸湿功能就是利用层间电荷的极化作用及多价阳离子的水合能力面发生多层吸水。该活性干燥剂具有充分的吸湿能力，无毒、无味，对金属无接触腐蚀、无环境污染、造价成本低等特点。除应用于军工部门外，还应用子仪器、仪表、电子器材、机械电气、医药食品、服装鞋帽、皮件皮具等行业[1]。

1.1 牌号及名称 M-83 型干燥剂 (M-83 蒙脱石)。袋装干燥剂按其用途分为三种类型: I 型——一般用途; II 型——无粉尘; III 型——特定用途。

# 可剥性塑料

# 1 概述

可剥性塑料是航空零部件用内包装材料。它是由塑料(或树脂)为主体成膜材料，加入缓蚀剂、增塑剂、稳定剂、防霉剂、矿物油等组成的，涂覆在制品表面上形成保护膜，不用时容易从制品表面剥去。具有涂覆简单、使用方便的特点。

可剥性塑料按它的组成不同，可分为溶剂型和热熔型两大类。热熔型可剥性塑料适用于外型较简单的贵重工具，如医用外科工具、齿轮等。对带有刃口的工具以及表面光洁度或精度较高的量具等的密封防锈更为适宜。溶剂型可剥性塑料适用于不易涂覆热熔型可剥性塑料的大型机械或大型部件。

1.1 牌号及名称 热熔型可剥性塑料、溶剂型可剥性塑料。  
1.2 基本组成[2] 热熔型可剥性塑料以纤维素塑料为基体，并与少量矿物油、增塑剂、树脂、稳定剂配制而成；溶剂型可剥性塑料以乙烯类树脂如聚氯乙烯、聚苯乙烯、过氯乙烯或它们与其他单体的共聚物为主体材料，配以颜料、增塑剂、稳定剂及溶剂等而成。

# 3.1 溶剂型可剥性塑料的涂覆方法

3.1.1 溶剂型可剥性塑料用每平方厘米 300 孔筛或多层纱布过滤, 除去杂质, 静置至无气泡后使用。  
3.1.2 表面准备 先将被涂覆物品的表面作清洁干燥处理，在表面上的凹处、小孔或缝隙处用铝箔或无腐蚀性的胶带包好或贴封，以便在去膜时易于剥除。若有会被溶剂破坏的组合材料，要用铝箔或胶带保护。

# 3.1.3 涂覆方式

（1）喷涂 将可剥性塑料放入加压槽，用干燥压缩空气喷涂。一次喷涂约 $0.2\mathrm{mm}$ 。如加热使加压

槽中材料温度为 $40\sim 50^{\circ}\mathrm{C}$ ，则涂膜干燥较快，一次喷涂可达 $0.3\sim 0.4\mathrm{mm}$ 。需要厚膜时，可增加喷涂次数。

（2）刷涂不用特殊设备，手工刷涂，故实际使用亦多。干燥较慢，且膜上常留有刷痕。  
（3）浸涂 适于小型物品涂覆。浪费及溶剂污染较小。  
（4）流涂或淋涂 用于大量连续涂覆，如金属板成型时就涂覆上溶剂型可剥性塑料。

# 3.2 热熔型可剥性塑料的涂覆方法

3.2.1 表面准备 准备浸涂的物品要预先做好清洁及干燥处理。物品上如有小孔、缝隙或其他深凹部分，用铝箔卷成筒堵塞，或用无腐蚀的胶带封闭。如果组合件上有非金属部分且易受此材料破坏的，用铝箔包好或用胶带封好。  
3.2.2 涂覆方式 热熔型可剥性塑料的涂覆有两种方法：一种是将物品直接浸入熔融的塑料液中涂覆；另一种是先用铝箔包裹物品，然后浸涂。在涂覆前要除去液面的气泡。

（1）浸渍涂覆较小的物品，可以一次浸涂。用金属丝、涂蜡的绳索或尼龙丝系好物品浸入熔融材料中 $3\sim 10$ s，提出后待冷却成膜。悬挂所用的丝或线要割断，并用熔融塑料将断头封于膜中，以免有缝隙渗入湿气。  
(2) 较长的物品宜分两次浸涂。第一次浸入涂覆大于物品长度一半的部分, 提出冷却后再浸涂另一半。两次浸涂的重叠处应不少于 $10 \mathrm{~mm}$ , 最好达 $20 \mathrm{~mm}$ 。  
（3）调节好浸涂的时间与熔融体的温度，以获得厚度为 $1.0\sim 2.0\mathrm{mm}$ 的膜层。
