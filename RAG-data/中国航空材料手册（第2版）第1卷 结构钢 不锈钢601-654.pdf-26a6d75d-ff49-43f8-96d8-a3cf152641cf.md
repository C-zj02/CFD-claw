# 1Cr18Mn8Ni5N

# 1 概述

1Cr18Mn8Ni5N是节镍奥氏体型不锈钢。该钢中以锰和氮代替部分镍，而获得较高的强度，但耐腐蚀性较1Cr18Ni9低。在 $800^{\circ}\mathrm{C}$ 以下有满意的抗氧化性及较高的中温强度。该钢可用于制造 $400^{\circ}\mathrm{C}$ 以下在大气或燃气产物中长期使用的航空发动机冲压和焊接构件，以及在 $800^{\circ}\mathrm{C}$ 以下大气或与燃烧产物相接触的短时使用零件。

# 1 概述

# 5 工艺性能与要求

5.1 成形性能 由于钢在熔炼或重熔过程中易产生铬和钛的氧化膜，使钢的铸造性能变坏，宜采用高温快铸工艺。铸温可选用 $1550\sim 1580^{\circ}\mathrm{C}$ ，铸型内钢液上升速度应两倍于普通珠光体钢，充填速度为碳钢的 $1\sim 1.5$ 倍。钢的收缩一般为 $2\% \sim 2.5\%$ 。  
5.2 焊接性能 焊接性能良好，可采用氩弧焊、点焊和缝焊。  
5.3 零件热处理工艺 $1080 \sim 1100^{\circ} \mathrm{C}$ 固溶，冷却介质可根据铸件大小、壁厚和形状复杂程度，采用水冷、油冷或气冷。当铸件含碳量为 $0.04\% \sim 0.06\%$ 时，其固溶温度应降低为 $950 \sim 1000^{\circ} \mathrm{C}$ 。根据使用要求，铸件可在固溶处理后加一道 $800^{\circ} \mathrm{C}$ 稳定化处理。
